    The local boundary package on G_{ℚ₂} (Stage C) #

    This file assembles the local boundary package boundaryPackage_GQ2 : BoundaryPackage GQ2Profinite — the G_{ℚ₂}-side instance of the uniform Stage-C datum of a tame quotient G_{ℚ₂} ↠ T_A, a maximal pro-2 quotient G_{ℚ₂} ↠ Π, and a common unramified character ν_loc : G_{ℚ₂} ↠ ℤ₂ through which both factor (manuscript reference/q2_manuscript.tex, Prop 3.14 prop:compatiblemarking, l.1085–1110; plan reference/plan_4turns.md Turn 4 §6).

    The two markings #

    • Pro-2 side (proved from the marked Demushkin normalization). The maximal pro-2 quotient map is the composite pro2Map_loc = maxPro2Proj G_{ℚ₂} ≫ (G_{ℚ₂}(2) ≅ D₀).hom ≫ (D₀ ≅ Π).hom, going from G_{ℚ₂} to its maximal pro-2 quotient G_{ℚ₂}(2) (maxPro2Proj), across Labute's classification (labute_GQ2_maxPro2_marked.iso), and along the profinite Nielsen comparison (D0_iso_PiGrp) to Π. Surjectivity is the composite of three surjections (maxPro2Proj_surjective and the two isomorphism hom-maps). Its unramified marking pro2Map_loc ≫ ν₂ = ν_loc is the chain marked (ν_ur^{loc} = iso.hom ≫ ν_ur) ∘ nu_ur_eq_hom_comp_nu_2 (ν_ur = D0_iso_PiGrp.hom ≫ ν₂); see pro2Map_loc_nu. The common character is ν_loc = maxPro2Proj G_{ℚ₂} ≫ ν_ur^{loc}.

      This is the local half of Prop compatiblemarking ("For the local source, use cor:relativeDemushkin", l.1106) and consumes the full marking carried by labute_GQ2_maxPro2_marked — exactly the load-bearing datum flagged in Remark rem:fullmarkingload (l.1113–1123): a bare abstract Demushkin iso is insufficient.

    • Tame side (one local axiom). The local tame quotient G_{ℚ₂} ↠ T_A with matching unramified character is Proposition 3.2 (prop:tamequotient, l.421: G_{ℚ₂}/W_F ≅ T_A) together with the local half of Prop compatiblemarking. This is genuine local class field theory — the tame quotient has pro-prime-to-2 cyclic inertia and procyclic unramified quotient, with geometric Frobenius acting on tame inertia by x ↦ x², i.e. τ^σ = τ² (l.421–434) — and is not available in mathlib v4.28.0. We therefore record it as the project's single new local axiom gq2_tame_quotient, a GQ2TameQuotient datum bundling the surjective tame map, the compatibility tameMap_loc ≫ ν_t = ν_loc (the same unramified character ν_loc as the pro-2 side), and the local tame maximality/factorization property used in the tame-frame exhaustion (Prop 3.2 + lem:o2tame on the local side).

    Axiom audit #

    boundaryPackage_GQ2 depends on exactly two non-standard axioms:

    • labute_GQ2_maxPro2_marked — the fully marked Labute comparison (Local/LabuteClassification.lean), reached through the pro-2 side;
    • gq2_tame_quotient — the local tame quotient introduced here.

    So #print axioms boundaryPackage_GQ2 = {propext, Classical.choice, Quot.sound, labute_GQ2_maxPro2_marked, gq2_tame_quotient}. (It does not depend on the Goursat core tameMap_ker_pro2Map_ge of Boundary/BoundaryPackage.lean — now proven unconditionally — which is a separate theorem unused here: assembling a BoundaryPackage value requires only its fields.)

        The tame side: one local axiom (Prop 3.2 + the local half of compatiblemarking) #

        Proposition 3.2 (prop:tamequotient, manuscript l.421) gives the canonical local tame quotient G_{ℚ₂}/W_F ≅ T_A; lem:o2tame identifies wild inertia with the local characteristic 2-core; and the local half of Prop compatiblemarking (l.1085–1110, "For the local source, use cor:relativeDemushkin") furnishes the unramified character matching the marked pro-2 quotient. These are local class field theory, irreducible and absent from mathlib v4.28.0; we package them as a single GQ2TameQuotient datum recorded by the axiom gq2_tame_quotient.
