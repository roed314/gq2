    The common tame quotient map Γ_A ⟶ T_A (Stage C, Proposition 3.2) #

    This file constructs the manuscript's common tame quotient surjection

    gammaTameMap : GammaA ⟶ Ttame

    i.e. the candidate-side half of Proposition 3.2 (reference/q2_manuscript.tex, prop:tamequotient, l.421–434): there is a canonical map Γ_A → T_A realising Γ_A/W_A ≅ T_A (with the wild generators x₀, x₁ killed). Together with Lemma 3.3 (lem:o2tame, l.436–446, O₂(Γ_A) = W_A) this exhibits T_A as the common tame quotient. The kernel characterisation ker = W_A is not proved here — it is left for the boundary package; this file delivers the map, its surjectivity, and the generator-image lemmas needed downstream.

    Method (universal property of the limit T_A) #

    This mirrors Boundary/CandidatePro2.lean's construction of p : Γ_A ⟶ Π, but on the tame side. We exhibit the canonical surjection sending σ ↦ tameSigma, τ ↦ tameTau, x₀ ↦ 1, x₁ ↦ 1 by assembling, for every tame index A : TameMarkedFinite, the per-coordinate surjection Γ_A ↠ Fin A.n supplied by the epimorphism semantics (GammaA_epi_equiv_admissible) applied to the tame marking σ ↦ A.s, τ ↦ A.t, x₀ ↦ 1, x₁ ↦ 1 of Fin A.n.

    The substantive new ingredient (relative to the pi side) is tameMarking_admissible: this marking is admissible. Generation is A.generates (the extra 1's do not enlarge the closure), the tame relator is A.tame, and — the crux — the wild relator vanishes: with x₀ = x₁ = 1 every wild-word atom collapses, the only surviving ω₂-power is (x_i τ)^{ω₂} = τ^{ω₂}, which is 1 because τ = A.t has odd order in a finite tame quotient (finite_tame_tau_odd A.tame), so powOmega2 (A.t) = 1 (powOmega2_eq_one_of_odd_order). This is the manuscript's "in the quotient by W_A the elements x₀, x₁ vanish and τ has odd order, so the wild relation disappears" (Prop 3.2 proof).

    The surjections form a cone over tameDiagram; gammaTameMap is the induced map into the limit T_A. It is surjective because its range is closed (continuous image of the compact Γ_A) and dense (it contains the topological generators tameSigma, tameTau of T_A).

      The tame marking of A is admissible (Proposition 3.2).

      • generates: A.s, A.t already generate (A.generates); the extra 1's do not enlarge the closure.
      • tameRel: reduces to rconj A.t A.s = A.t², which is A.tame.
      • wildRel: with x₀ = x₁ = 1 the wild word collapses to a product of ω₂-powers of τ = A.t; since A.t has odd order in the tame quotient (finite_tame_tau_odd A.tame), powOmega2 (A.t) = 1, so the relator vanishes.
      • x₀, x₁ ∈ O₂: both images are 1 ∈ twoCore.
