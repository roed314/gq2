    Boundary-framed marked targets and the exact-image inheritance lemma (Stage D) #

    This file realises manuscript Definition 4.1 (def:framed) and the structural content of Theorem 4.2 (thm:fixedframe) that is finite-group-theoretic and Q₂-independent: the boundary-framed marked targets, the boundary frame (α, ψ, β) cutting a finite target down to a fixed tame quotient H and a fixed elementary decoration E, the boundary-framed surjection count boundaryFramedSurj, and the exact-image inheritance lemma (exactImageTarget / exactImageTarget_boundaryFramed).

    We deliberately do not prove the global induction of Theorem 4.2 (that is Stage H — it needs the linear lifting theory, exact-image recursion, Fourier inversion and strict decrease of marked 2-kernels). We only:

    • define the data structures (BoundaryFramedTarget, BoundaryFrame, boundaryFramedSurj);
    • prove that an exact-image stratum inherits the boundary-framed structure.

    Manuscript Definition 4.1 (def:framed) #

    A boundary-framed marked target is a tuple 𝒴 = (Y, L_Y, π_Y, θ_Y) where L_Y ◁ Y is a finite 2-group, π_Y : Y ↠ H has kernel L_Y, and θ_Y : Y → E is a homomorphism. Put q_Y = (π_Y, θ_Y) : Y → H × E.

    The frame fixes, via α : 𝕋_tame ↠ H and ψ̄ : Π → E, the induced β : ∂_bd → H × E, β(t, p) = (α t, ψ̄ p) (manuscript eq:beta). We compare only surjections f : Γ ↠ Y whose composite q_Y ∘ f equals β ∘ b_Γ (eq:eGamma): the boundary-framed count.

    Manuscript exact-image inheritance (def:framed, paragraph after eq:eGamma) #

    An exact-image stratum consists of maps whose image is a specified subgroup J, rather than maps that merely land in J. If J ≤ Y projects onto H, then 𝒥 = (J, J ∩ L_Y, π_Y|_J, θ_Y|_J) inherits the same structure.

    exactImageTarget builds 𝒥 as a genuine BoundaryFramedTarget — so the fact that it type-checks is exactly the inheritance — and exactImageTarget_boundaryFramed exposes the three substantive inherited facts (J ∩ L_Y is normal, is a 2-group, and is the kernel of the restricted π_Y|_J). Because H and E are preserved on the nose, the very same BoundaryFrame frames the stratum (exactImageFrame).

    A boundary-framed marked target 𝒴 = (Y, L_Y, π_Y, θ_Y) (manuscript Definition 4.1, def:framed): a finite group Y with a normal finite 2-subgroup L_Y, a surjection π_Y : Y ↠ H onto a finite tame quotient with kernel exactly L_Y, and an elementary decoration θ_Y : Y → Multiplicative E into the multiplicative image of a finite 𝔽₂-vector space E. The pair q_Y = (π_Y, θ_Y) fixes both the tame quotient Y / L_Y ≅ H and the decoration.

                def Q2Presentation.BoundaryFramedTarget.exactImageTarget (Yt : BoundaryFramedTarget) (J : Subgroup Yt.Y) (hJ : Function.Surjective ⇑(Yt.piY.comp J.subtype)) :
                BoundaryFramedTarget

                The exact-image stratum 𝒥 = (J, J ∩ L_Y, π_Y|_J, θ_Y|_J) (manuscript Definition 4.1, exact-image paragraph), as a genuine BoundaryFramedTarget.

                The hypothesis hJ says J projects onto H (π_Y|_J surjective), which is the manuscript's "J ≤ Y projects onto H". All four boundary-framed axioms are inherited:

                • J ∩ L_Y is normal in J (Normal.subgroupOf);
                • J ∩ L_Y is a 2-group (subgroupOf_LY_isTwoGroup);
                • π_Y|_J : J ↠ H is surjective (the hypothesis hJ) with kernel J ∩ L_Y (ker_piY_comp_subtype).

                The tame quotient H and the elementary decoration E are preserved on the nose. This is the inheritance lemma: it being well-typed is the content.

                  Exact-image inheritance lemma (manuscript Definition 4.1: "If J ≤ Y projects onto H, then 𝒥 = (J, J ∩ L_Y, π_Y|_J, θ_Y|_J) inherits the same structure"). The exact-image stratum is again boundary-framed: its marked normal subgroup J ∩ L_Y is normal and a 2-group, and is exactly the kernel of the restricted tame quotient π_Y|_J : J ↠ H.
