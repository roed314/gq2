    The profinite pro-2 group Π and its unramified marking ν₂ (Stage C, work order C3) #

    This file realises the manuscript's maximal pro-2 quotient Π = ⟨σ, x₀, x₁ ∣ x₀^{σ²} x₀ [x₁,σ] = 1⟩_{pro-2} (reference/q2_manuscript.tex, Proposition 3.10 prop:pro2, lines ~962–995) as the cofiltered inverse limit, in ProfiniteGrp, of all finite marked pro-2 boundary quotients — exactly mirroring the Stage-C construction of the tame group Ttame in Boundary/Ttame.lean, but with the 3-generator pro-2 marking (σ, x₀, x₁) (carrying the 2-group side condition) in place of the 2-generator tame marking (s, t).

    The finite index objects PiMarkedFinite, their morphisms PiHom, the index category and PiHom_ext are reused from Boundary/Nielsen.lean (work order C4); this file only adds the cofiltered structure on that category and the inverse-limit construction on top of it.

    Main contents:

    • IsCofiltered PiMarkedFinite — binary subdirect objects (piSubdirectObj) plus the subsingleton hom-sets (PiHom_ext) make the index category cofiltered.
    • piDiagram : PiMarkedFinite ⥤ ProfiniteGrp, PiGrp := Limits.limit piDiagram, the bridge piIso / PiGrp_ext, the universal generators piSigma, piX0, piX1 : PiGrp, their projection-eval lemmas, the coordinatewise boundary relation PiGrp_piRel, and dense generation piGen_topologically_generate.
    • nu_2 : PiGrp ⟶ ProfiniteZ2 — the unramified character σ ↦ 1, x₀ ↦ 0, x₁ ↦ 0 (eq:nu2). In a pro-2 group the wild generators x₀, x₁ are killed in the unramified quotient; only the procyclic σ-direction survives. Built as the map into ℤ₂ = lim_n ZMod(2ⁿ) whose n-th layer is the projection of PiGrp through the auxiliary cyclic-2-group boundary quotient cyclicPi n (carrier C2Pow n, σ = additive generator, x₀ = x₁ = 1).

    Instance gotcha (inherited from AdmissibleIndex.lean / Nielsen.lean) #

    Fin n carries the global Fin.instMul, not the transported group structure A.groupFin. We thread A.groupFin explicitly (@, letI, projection access) through every Fin-level construction so Fin.instMul is never synthesized.
