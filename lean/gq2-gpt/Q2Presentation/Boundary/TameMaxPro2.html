    The maximal pro-2 quotient of the tame group is ℤ₂ (Stage C) #

    This file identifies the maximal pro-2 quotient of the profinite tame group Ttame with the unramified ℤ₂ (manuscript reference/q2_manuscript.tex, Lemma 3.1 lem:tamefinite, Prop 3.2 prop:tamequotient; plan reference/plan_4turns.md Turn 4 §2).

    The unramified character nu_t : Ttame ⟶ ProfiniteZ2 (σ ↦ 1, τ ↦ 0) exhibits ℤ₂ as the maximal pro-2 quotient: it is surjective, ℤ₂ is pro-2, and every finite 2-group quotient of Ttame kills τ (Lemma 3.1: τ has odd order, hence = 1 in a 2-group), so the quotient factors through the unramified ⟨σ⟩ = ℤ₂ direction. Combining the universal property of MaximalPro2Quotient both ways gives the isomorphism Ttame_maximal_pro2.

    Maximality: ker nu_t is contained in every pro-2 index (Lemma 3.1) #

    source
    theorem Q2Presentation.nu_t_ker_le (U : Pro2Index Ttame) {x : ↑Ttame.toProfinite.toTop} (hx : (ProfiniteGrp.Hom.hom nu_t) x = 1) :
    x ∈ ↑U.obj.toOpenSubgroup

    The maximality core. If nu_t x = 1 then x lies in every pro-2 index U of Ttame. Each finite 2-group quotient F = Ttame ⧸ U is cyclic generated by the image y = mk σ (because τ ↦ 1, tameTau_mem_pro2), so the quotient map factors through the k-th unramified layer (k with Nat.card F = 2ᵏ): mk = ψ ∘ tameLayerHom k, where ψ : C2Pow k →* F sends the additive generator to y. Since tameLayerHom k x = z2Projection k (nu_t x) = 1, we get mk x = ψ 1 = 1, i.e. x ∈ U.

    The identification MaximalPro2Quotient Ttame ≅ ℤ₂ #

    source
    noncomputable def Q2Presentation.Ttame_maximal_pro2 :
    MaximalPro2Quotient Ttame ≅ ProfiniteZ2

    The maximal pro-2 quotient of the tame group is the unramified ℤ₂ (manuscript Lemma 3.1 + Prop 3.2). The unramified character nu_t lifts (since ℤ₂ is pro-2) to a canonical comparison g : MaximalPro2Quotient Ttame ⟶ ℤ₂; it is surjective because nu_t is, and injective by the maximality core nu_t_ker_le (every finite 2-group quotient of Ttame kills τ, hence factors through the unramified line), so it is an isomorphism.
