    The profinite tame group Ttame and its unramified marking (Stage C, work order C2) #

    This file realises the manuscript's finite-quotient tame group T_A = ⟨σ, τ ∣ τ^σ = τ²⟩_prof (reference/q2_manuscript.tex, l.401, Lemma 3.1 lem:tamefinite, Prop 3.2 prop:tamequotient, Lemma 3.3 lem:o2tame) as the cofiltered inverse limit, in ProfiniteGrp, of all finite marked tame quotients — exactly mirroring the Stage-B construction of Γ_A in Candidate/AdmissibleCat.lean + Candidate/GammaA.lean, but with the 2-generator tame marking (s, t) in place of the 4-generator admissible marking q : Gen → Fin n.

    Main contents:

    • TameMarkedFinite — a concrete tame-marked finite group on the skeleton Fin n (generators s, t, generating, with rconj t s = t²).
    • TameHom + Category TameMarkedFinite — generator-preserving homs and the index category; tameHom_ext (any two parallel arrows are equal, the marking generates), binary subdirect objects (tameSubdirectObj), and IsCofiltered TameMarkedFinite.
    • tameDiagram : TameMarkedFinite ⥤ ProfiniteGrp, Ttame := Limits.limit tameDiagram, the universal generators tameSigma, tameTau : Ttame, their projection-eval lemmas, the coordinatewise tame relation Ttame_tameRel, and dense generation tameGen_topologically_generate.
    • nu_t : Ttame ⟶ ProfiniteZ2 — the unramified character σ ↦ 1, τ ↦ 0 (τ is pro-odd, killed in the unramified pro-2 quotient). Built as the map into the limit ℤ₂ induced by the per-layer projections through the auxiliary cyclic 2-group tame quotients cyclicTame n (carrier C2Pow n, s = additive generator, t = 1). This realises Lemma 3.1's "every normal 2-subgroup is central + unramified" structurally: the only pro-2 information in a tame quotient is the unramified σ-direction.

    Instance gotcha (inherited from AdmissibleIndex.lean) #

    Fin n carries the global Fin.instMul, not the transported group structure A.groupFin. We thread A.groupFin explicitly (@, projection access) through every Fin-level construction so Fin.instMul is never synthesized.

    A concrete tame-marked finite group with carrier the skeleton Fin n: size n, nonemptiness 0 < n, group structure groupFin, two marked generators s, t, a proof they generate, and the tame relation τ^σ = τ² (rconj t s = t²).

    • s : Fin self.n

      The marked tame inertia-free generator σ.

    • t : Fin self.n

      The marked tame inertia generator τ.

    • generates : Subgroup.closure {self.s, self.t} = ⊤

      s, t generate the carrier.

    • tame : rconj self.t self.s = self.t ^ 2

      The tame relation τ^σ = τ².
