    The admissible index category and its cofilteredness (Stage B, work order B2) #

    This file upgrades AdmissibleIndex (the skeletal admissible marked finite groups of Candidate/AdmissibleIndex.lean) to a category whose morphisms are the generator-preserving group homomorphisms, and proves that category is cofiltered.

    Cofilteredness is the Lean form of manuscript Lemma 2.1 (finite subdirect closure) and Lemma 2.2 (cofinality): every finite diagram in AdmissibleIndex has a cone, built from the single uniform subdirect construction admissible_subdirect. Two structural facts make this clean:

    • Hom uniqueness (admissibleHom_ext): a morphism is pinned by its action on the marking, and the marking generates the source (AdmissibleIndex.admissible.1 : generates A.q), so any two parallel arrows coincide. The index category has no nontrivial parallel-arrow coherence — equalizers are free, and the cone_maps half of cofilteredness is trivial.
    • Binary cones (subdirectObj): the skeletalized diagonal subdirect quotient of the family ![A, B] admits coordinate projections to A and B.

    GammaA is built (in B3) as the cofiltered limit of this category.

    Instance gotcha (inherited from AdmissibleIndex.lean) #

    Fin n carries the global Fin.instMul (mod-n multiplication), which is not definitionally equal to the group multiplication transported by skeletalize. Whenever a Group/Mul (Fin n) is needed we thread the bundled AdmissibleIndex.groupFin explicitly (@, projection access) so Fin.instMul is never synthesized — otherwise instance synthesis reports a non-defeq clash and the elaborator stalls on isDefEq heartbeats.

          theorem Q2Presentation.admissibleHom_ext {A B : AdmissibleIndex} (f g : A ⟶ B) :
          f = g

          Uniqueness of admissible morphisms. Both maps send the marking A.q to the same values (B.q, by map_q), and A.q generates A (A.admissible.1), so they agree on a generating set and hence everywhere (MonoidHom.eq_of_eqOn_dense). Consequently the index category has no nontrivial parallel arrows.

            Transporting homomorphisms out of a skeletalization #

            A morphism out of a skeletalized object is built from a marking-preserving homomorphism out of the original group G, precomposed with the transport equivalence Fin (skeletalize G q h).n ≃* G. The group instance on the skeleton is threaded explicitly so the global Fin.instMul is never synthesized.
