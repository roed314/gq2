    Proof outline (= manuscript Lemma 2.2 compactness argument) #

    Work in the explicit inverse-limit model L := ProfiniteGrp.limit admissibleDiagram (via the bridge gammaIso), whose points are compatible families m : ∀ A, Fin A.n carrying the subspace topology of the product ∏_A Fin A.n.

    • limit_finite_factor: a continuous map f : L → H into any discrete space depends on a single coordinate. Since H is discrete, each fibre f⁻¹{f m} is open; in the product topology it contains a basic-clopen box fixing finitely many coordinates J_m (isOpen_pi_iff), on which f is constant. Compactness of L (IsCompact.elim_finite_subcover) reduces the cover {box_m} to a finite subcover, so f is determined by the finite union of coordinates ⋃ J_{m₀}; cofiltered domination (exists_dominating, folding IsCofiltered.min) collapses that finite set to one coordinate B. This mirrors GammaA.dense_gammaGenFamily.

    • piMon_surjective: each limit projection π_A : Γ_A → Fin A.n is surjective (the universal generators map onto the markings A.q, which generate Fin A.n by admissibility — mirrors GammaA.map_evalAt_closure_eq_top).

    • GammaA_map_to_finite_factors: transport the single-coordinate factorization to Γ_A, then lift φ over the surjection π_A with MonoidHom.liftOfSurjective (kernel containment ker π_A ≤ ker φ holds because φ is constant on π_A-fibres). The lifted monoid hom is automatically continuous (Fin A.n discrete), giving the factoring morphism ψ.

      theorem Q2Presentation.GammaA_map_to_finite_factors {H : Type} [Group H] [Finite H] (φ : GammaA ⟶ ProfiniteGrp.ofFiniteGrp (FiniteGrp.of H)) :

      Finite factorization (manuscript Lemma 2.2, plan §5 / work order B4).

      Every continuous homomorphism φ : Γ_A ⟶ H to a finite group factors through one of the finite admissible quotients: there is an admissible index A and a (continuous) homomorphism ψ : Fin A.n ⟶ H with φ = π_A ≫ ψ.

      The proof transports the single-coordinate factorization limit_finite_factor across the bridge gammaIso, then lifts φ over the surjection π_A (piMon_surjective) using MonoidHom.liftOfSurjective: the kernel containment ker π_A ≤ ker φ holds because φ is constant on π_A-fibres. The lifted monoid hom is continuous for free (Fin A.n is discrete), giving ψ.
