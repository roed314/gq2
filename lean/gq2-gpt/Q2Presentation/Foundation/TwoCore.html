    The p-core and the 2-core O₂(G) (Stage A) #

    This file builds the machinery of the p-core Oₚ(G) — the largest normal p-subgroup of G — and specialises it to the 2-core O₂(G), the wild subgroup that controls the Q₂ presentation.

    Mathlib (as of this toolchain) has no pCore / Fitting-subgroup API, so we define it here as the join (sSup) of all normal p-subgroups and develop the interface used by the rest of the formalisation:

    • pCore, twoCore — the definitions;
    • pCore_normal / twoCore_normal — normality (any group);
    • pCore_isPGroup — for a finite group the join really is a p-group (the substantive lemma: a finite directed join of normal p-subgroups, proved by Finset.sup_induction);
    • normal_pSubgroup_le_pCore — maximality;
    • mem_twoCore_iff_normal_closure_is_two_group — the characterisation used in Proposition 2.3 (current_manuscript, §2): a single element lies in O₂(G) iff the normal closure it generates is a 2-group;
    • hom_image_twoCore_le_twoCore — functoriality under surjections (the epimorphism sends the wild subgroup into the target 2-core);
    • quotient_by_twoCore_has_no_nontrivial_normal_two_subgroup — O₂(G/O₂(G)) = 1, the fact that makes the wild subgroup characteristic (Lemma 3.3 / lem:o2tame).

    All proofs are constructive and sorry-free; the p-group facts come from Mathlib.GroupTheory.PGroup.

    def Q2Presentation.pCore (p : ℕ) (G : Type u_1) [Group G] :
    Subgroup G

    The p-core Oₚ(G): the join of all normal p-subgroups of G.

    For a finite group this is the unique largest normal p-subgroup (pCore_isPGroup, normal_pSubgroup_le_pCore). The definition itself makes sense for any group, and normality holds unconditionally (pCore_normal).

        theorem Q2Presentation.pCore_mem {p : ℕ} {G : Type u_1} [Group G] [Finite G] :
        (pCore p G).Normal ∧ IsPGroup p ↥(pCore p G)

        For a finite group, the p-core is itself a normal p-subgroup.

        The join of all normal p-subgroups is a finite supremum (there are finitely many subgroups), and the class of normal p-subgroups is closed under binary join (IsPGroup.to_sup_of_normal_right + Subgroup.sup_normal) and contains ⊥; Finset.sup_induction then propagates the property to the whole join.

        theorem Q2Presentation.pCore_quotient_eq_bot {p : ℕ} {G : Type u_1} [Group G] [Finite G] :
        pCore p (G ⧸ pCore p G) = ⊥

        Oₚ(G / Oₚ(G)) = 1: the quotient by the p-core has no nontrivial normal p-subgroup. Equivalently, the p-core is idempotent in the sense that passing to the quotient annihilates it.
