    Candidate actual obstruction/Frattini proof refinements #

    This sidecar targets the two current Section 8 actual candidate packets introduced in CandidateMiddleFrattiniActualProofs.lean:

    • centralCover_candidateActualMiddleRLiftObstructionMapData;
    • centralCover_candidateActualFrattiniAffineFiberExactImageEquivData.

    It intentionally does not import that file. Instead it proves route-ready replacement theorems from the smaller packets already isolated lower in the candidate exact-image stack:

    • the middle exact-image R-lift obstruction map centralCover_candidateMiddleRLiftObstructionMapData_residual;
    • the Frattini exact-image partition centralCover_candidateFrattiniAffineFiberPartition_residual.

    Thus the route below has no dependency on the two old actual packet axiom names. No routed/root file is edited here.
