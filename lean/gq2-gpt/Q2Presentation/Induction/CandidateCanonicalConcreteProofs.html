    This sidecar narrows the candidate Section 8 final-closure packet centralCover_candidateCanonicalConcreteProofData.

    The manuscript shape is the canonical exact-image tower Y -> B = Y/R -> C = Y/K. The theorem below derives the current aggregate canonical candidate packet from the three smaller component packets already isolated in the tree:

    • the actual quotient-frame postcomposition map;
    • the middle R-lift obstruction on the canonical exact-image source;
    • the Frattini exact-image affine-fibre equivalence for that same obstruction.

    No routed file is edited here. The last theorem gives the final count route with the same extraspecial determinant refinement as the current route, but with the candidate call replaced by the narrowed canonical concrete theorem.
