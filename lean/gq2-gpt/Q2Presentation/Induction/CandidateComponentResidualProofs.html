    Candidate component residual proofs #

    This sidecar strictly narrows the three current candidate Section 8 component residuals used by CandidateCanonicalConcreteProofs.lean.

    The manuscript tower is the canonical exact-image tower

    Y -> B = Y/R -> C = Y/K.

    For the quotient-frame component, the canonical framed quotient construction already preserves the boundary codomain: the middle quotient has the same tame quotient H, the same decoration module E, and the same boundary frame beta. The quotient-frame map is therefore theorem-level for that explicit canonical construction.

    For the source-side components, the middle R-obstruction and its Frattini exact-image affine-fibre equivalence are narrowed to one coherent packet. This avoids asking for a Frattini equivalence for an arbitrary independently chosen obstruction evaluation.

    No routed file is edited here.
