    Concrete candidate exact-image closers #

    This sidecar pushes the four candidate-side Section 8 packets from CandidateExactImagePacketClosers.lean down to smaller theorem-facing data.

    The split follows reference/q2_manuscript.tex, Section 8:

    • the finite quotient map Y -> B = Y/R supplies the profinite postcomposition morphism; only the boundary-frame compatibility remains as semantic data (lem:covertransform);
    • the exact-image base index can be the finite set of middle boundary-framed exact-image maps itself (def:Xgamma);
    • the obstruction map is an explicit evaluation on that finite index (lem:obstructionseparation);
    • the Frattini/cardinality packet follows from an actual exact-image equivalence/partition (prop:finalfourier), rather than a bare cardinal equality.

    No theorem below uses the four packet axioms from CandidateExactImagePacketClosers.lean.
