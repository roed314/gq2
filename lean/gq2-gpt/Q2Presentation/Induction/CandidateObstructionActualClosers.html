    Actual candidate obstruction closers #

    This sidecar attacks the three semantic candidate residuals currently isolated in CandidateObstructionSemanticClosers.lean.

    The file does not edit the routed integration files. It proves the current semantic residual conclusions from narrower manuscript-facing packets already present in the sidecar stack:

    • actual quotient postcomposition is obtained from the quotient-frame commuting square for the concrete map T.topToMiddle : Y -> Y/R;
    • the middle exact-image obstruction evaluation is obtained from the actual R-lift obstruction map on the canonical X_Gamma(B) index;
    • the top affine-fibre/Frattini equivalence is obtained from the Frattini-cover exact-image equivalence for the canonical affine fibres.

    Thus the theorem closures below avoid the three axioms named in CandidateObstructionSemanticClosers.lean; the remaining axioms are the smaller packets that still encode the manuscript's central-cover/lift-torsor semantics.
