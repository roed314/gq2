    Canonical candidate obstruction concrete proofs #

    This sidecar keeps the candidate Section 8 obstruction data tied to the same canonical target quotient tower used by the target/proper-image route. It is intended as a route-ready replacement for the three live candidate residual axioms on an arbitrary elementary tower: quotient-frame compatibility, the middle R-lift obstruction map, and the Frattini cover equivalence.

    The new residual below is indexed by D : CentralCoverTargetCanonicalQuotientTowerData B, so the middle source is the canonical exact-image source for D.tower, the top-to-middle map is the actual quotient-frame map of that tower, and the Frattini datum is an actual exact-image equivalence for the canonical affine fibres attached to the same middle obstruction.

    Candidate obstruction data for the canonical quotient tower D.tower.

    This is deliberately not a packet for an arbitrary bare elementary tower. The three fields are the manuscript-facing objects for the canonical tower:

    • quotient-frame commutation for the actual top-to-middle quotient map;
    • the R-valued obstruction on the canonical middle exact-image source;
    • the top exact-image/affine-fibre equivalence for the canonical affine fibres.
