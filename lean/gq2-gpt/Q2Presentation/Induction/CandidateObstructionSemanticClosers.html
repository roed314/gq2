    Candidate obstruction semantic closers #

    This sidecar narrows the current Section 8 candidate residuals to the actual semantic objects used by the final route:

    • quotient-frame data is the real postcomposition by the tower map T.topToMiddle : Y -> B = Y/R; the only residual field is boundary preservation for that postcomposition;
    • the R-lift obstruction is an evaluation on the canonical middle exact-image source X_Gamma(B) from def:Xgamma;
    • the Frattini packet is an explicit top exact-image equivalence for the canonical affine fibres, not merely an anonymous partition datum.

    No theorem below calls the older residuals centralCover_candidateQuotientFrameMapData_residual, centralCover_candidateMiddleRLiftObstructionMapData_residual, or centralCover_candidateFrattiniAffineFiberPartition_residual.
