    Candidate Route114 semantic closers #

    This sidecar targets the two candidate Section 8 leaves exposed by Route114:

    • centralCover_candidateMiddleScalarObstructionEvaluatorData;
    • centralCover_candidateFrattiniAffineFiberCoordinateData.

    There is no proof of these facts from already-closed Lean data in the current API. The useful lowering is to the coherent constructor shapes isolated in CandidateScalarFrattiniCurrentProofs:

    • one selected scalar-pushout evaluator, with linearity for that same choice;
    • one selected Frattini coordinate/realization tuple, with inverse laws for that same pair and the selected middle obstruction.

    The route-ready provider below has the same call shape as the Route114 candidate provider, but its axiom closure contains those coherent constructor packets instead of the two Route114 semantic leaves.
