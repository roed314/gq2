    Candidate Route124 exact-image closers #

    This sidecar is a Route124-named wrapper around the already verified current semantic exact-image split from CandidateRoute113ExactImageClosers.

    The current public candidate branch calls centralCover_candidateLiftPartitionData_from_exactImageFinalClosers, whose closure still contains the two exact-image residual atoms

    • centralCover_candidateMiddleExactImageObstructionEvaluationData_residual;
    • centralCover_candidateTopExactImageAffineFiberEquivData_residual.

    The theorems below keep the same downstream consumer shape but route the candidate packet through the smaller current semantic packets exposed by CandidateCurrentPacketSplitClosers:

    • centralCover_candidateMiddleScalarObstructionEvaluatorData;
    • centralCover_candidateFrattiniAffineFiberCoordinateData.

    No theorem below calls either exact-image residual axiom directly, and no local/CFT axiom is touched.
