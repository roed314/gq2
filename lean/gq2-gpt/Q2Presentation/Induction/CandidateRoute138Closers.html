    Candidate Route138 closers #

    This sidecar attacks only the Route138 candidate Section 8 leaves. The Route138 integrated proof uses the Route115 scalar/Frattini split:

    • centralCover_candidateMiddleScalarPushoutValueData;
    • centralCover_candidateMiddleScalarPushoutLinearityData;
    • centralCover_candidateFrattiniLiftCoordinateData;
    • centralCover_candidateFrattiniRealizationData;
    • centralCover_candidateFrattiniCoordinateInverseData.

    The arbitrary-linearity and arbitrary-inverse leaf shapes are stronger than the manuscript recursion needs. Section 8 uses a selected middle R-obstruction map on the canonical exact-image index and the matching Frattini exact-image partition. This file routes the candidate contribution through those narrower packets:

    • centralCover_candidateMiddleRLiftObstructionMapData_residual;
    • centralCover_candidateFrattiniAffineFiberPartition_residual.

    It leaves all public route files untouched.
