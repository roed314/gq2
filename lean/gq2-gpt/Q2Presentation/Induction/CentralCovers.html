    The §8 conversion, faithfully (manuscript sec:fourier) #

    prop:finalfourier (eq:recursionR1, l.4259) reads

      e_Γ(Y) = (z_R/|𝒳_R|) · Σ_{λ∈𝒳_R} (2·m_{Γ,λ}(B) − e_Γ(B)),     m_{Γ,0}(B) = e_Γ(B),

    and its proof is the lift-torsor decomposition: for a fixed exact-image map to B the set of R-lifts, when the R-obstruction vanishes, is a torsor of size z_R = |Z¹_{Γ,ρ}(R)| (lem:affinelifting, eq:mumultiplicity; Lifting.lift_torsor_card), and every such lift is automatically surjective onto Y because R = Φ(K) is the Frattini subgroup (prop:finalfourier). Hence the boundary-framed surjections onto Y are, source by source, the disjoint union (a dependent Σ-type) over the unobstructed exact-image base maps {x : X_Γ // o_Γ x = 0} of the R-lift torsor:

      boundaryFramedSurj(Y)  ≃  Σ x : {x : X_Γ // o_Γ x = 0}, Lift_Γ x,

    each Lift_Γ x an AddTorsor under the R-cocycle group Z¹_Γ(R) with |Z¹_Γ(R)| = z_R. We record exactly this torsor decomposition per source (CoverRealization.bijA/bijQ over ZA/LiftA, ZQ/LiftQ), so the realize_Γ numerator e_Γ(Y) = z_R·#{o_Γ = 0} (prop:finalfourier) is PROVEN here — not assumed — by the lift-torsor count Lifting.lift_torsor_card (framedCount_eq_zR_of_torsorDecomp) in realizeA_of/realizeQ_of.

    Part 1 — The §8 central-cover realization over a minimal block #

    CoverRealization B is the source-uniform §8 output over a minimal non-scalar block B : MinimalBlock p. It carries exactly the pre-Fourier obstruction realization of prop:finalfourier/lem:properimagesubtraction/lem:phasecover — relative to the block, so the obstruction space W = B.obstructionSpace, the exponent B.dimExp and the lift-torsor size B.zR are inherited (not re-posited).
