    Elementary-stage residuals for Section 8 #

    This sidecar refines the two elementary-stage residuals in CentralCoverFineResiduals.lean.

    The manuscript source is Section 8, especially eq:Mstage, eq:Mdescent, and lem:strictdecrease. The existing MinimalBlock p abstraction contains the module-level block data R, M, C, but it does not contain the realized normal subgroups R <= K <= Y, the quotient targets B = Y/R, C = Y/K, or the proper exact-image subgroups J < B. Consequently the old residuals

    • centralCover_elementaryLowerTarget;
    • centralCover_elementaryProperImageChildrenData;

    cannot be manuscript-faithfully proved from the current fields alone. The packets below isolate exactly that missing finite group data, then reassemble the existing coarse elementary-stage interfaces from it.
