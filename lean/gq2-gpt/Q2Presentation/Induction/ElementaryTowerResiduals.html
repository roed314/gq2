    Target-tower residual split for Section 8 #

    This sidecar refines the two remaining target-side elementary-stage packets in ElementaryStageResiduals.lean.

    Manuscript anchors:

    • eq:targettower: R = Phi(K) <= K <= L_Y, with quotient maps Y -> B = Y/R -> C = Y/K;
    • eq:Mstage / eq:Mdescent: proper exact-image subgroups J < B mapping onto C, with |J ∩ L_B| = |J ∩ M| |L_C| and 2 |J ∩ M| <= |M|.

    The current MinimalBlock API is module-level: it supplies R, M, T, the simple head, and obstruction-space data, but it still does not construct the realized normal subgroups and quotient maps inside an arbitrary finite target. Accordingly this file does not claim a full de-axiomatization. It strictly refines the two live target residuals into concrete finite group packets and proves the exact reassembly theorems needed by the existing elementary-stage interfaces.

    Type

    The subgroup part of the manuscript target tower.

    This is the concrete finite-group content before quotient targets are attached: R = Phi(K), R <= K <= L_Y, both normal in Y, and R nontrivial so that the middle quotient is strictly smaller.

    • Rsub : Subgroup p.fst.Y

      The realized subgroup R = Phi(K) of the original finite target Y.

    • Ksub : Subgroup p.fst.Y

      The realized subgroup K of the original finite target Y.

    • Rsub_normal : self.Rsub.Normal

      R is normal in Y.

    • Ksub_normal : self.Ksub.Normal

      K is normal in Y.

    • Rsub_le_Ksub : self.Rsub ≤ self.Ksub

      Manuscript tower condition R <= K.

    • Ksub_le_LY : self.Ksub ≤ p.fst.LY

      Manuscript tower condition K <= L_Y.

    • Rsub_card_gt_one : 1 < Nat.card ↥self.Rsub

      The realized R is nontrivial.

                Properness of J gives 2 |J ∩ M| <= |M|.
