    Integrated current route, fourteenth refinement #

    Route14 is the first fully-composed route after the route12/route13 sidecar batch. It keeps the same finite-count theorem shape as route13, but routes the nonterminal Fourier branch through the strongest current manuscript-shaped providers:

    • candidate scalar/Frattini data via CandidateScalarFrattiniSemanticRoute12Proofs;
    • target same-tower quotient/action data via TargetKmodRSameTowerActualRoute12Proofs;
    • kernel/Frattini lower tower data via MinimalBlockKernelFrattiniCoherentRoute13Proofs;
    • head/invariant data via MinimalBlockHeadInvariantSemanticRoute12Proofs;
    • determinant/pushout data via MinimalBlockCentralPushoutRoute12Proofs.

    This intentionally exposes finer non-local packet leaves in the final axiom closure instead of hiding them behind older coarse residual names.
