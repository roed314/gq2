    Integrated current route, second refinement #

    This sidecar keeps IntegratedCurrentRouteProofs.lean stable and combines two post-checkpoint refinements whose source files intentionally import the current route:

    • CandidateScalarFrattiniActualClosers replaces the five candidate scalar-pushout / Frattini-coordinate packets by lower §8 packets.
    • MinimalBlockSimpleHeadScalarClosers replaces the four simple-head/scalar packets by smaller manuscript-shaped semantic packets.
    • MinimalBlockExtraspecialClosers3 replaces the five extraspecial determinant packets by one coherent actual central-pushout/extraspecial construction packet.
