    Actual central-extension pushout packet split #

    This sidecar further decomposes the current Section 7 final-route residual

    • minimalBlock_preheadActualCentralExtensionPushoutData.

    Manuscript anchor: reference/q2_manuscript.tex, Section 7, eq:targettower, lem:simplehead, lem:collapse, and prop:simpleheaddet. For every invariant nonzero scalar character lambda : R -> F_2, the paper forms the quotient central extension K_lambda = K / ker lambda, with self-duality pairing, square map, diagonal and polar identities, naturality, annihilation on T, and radical containment.

    MinimalBlockCentralPushoutFinalClosers packages all of that as one residual. The split below keeps the same semantic target but separates it into narrower, same-family packets:

    • PhiData: the scalar pushout pairing phi_lambda;
    • SquareData: the actual square map q_lambda;
    • DiagonalData: q_lambda(m) = phi_lambda(m,m);
    • PolarData: the polar form is the symmetrized pairing;
    • NaturalityData: equivariance of phi_lambda;
    • AnnihilationData: q_lambda|_T = 0 and b_lambda(T,M)=0;
    • RadicalContainmentData: the polar radical is contained in T.

    The theorem minimalBlock_preheadActualCentralExtensionPushoutData_from_splitPackets reassembles the old broad residual from these smaller packets without using the old broad residual axiom.
