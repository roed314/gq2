    Actual determinant packet from semantic simple-head pieces #

    This sidecar leaves the routed files untouched and narrows the current residual minimalBlock_preheadCentralPushoutSimpleHeadDeterminantActualData.

    For a fixed actual semantic central-pushout family S, the current residual is the bundled prop:simpleheaddet packet. The split packet below records the four manuscript-shaped pieces already exposed by the semantic §7 API for the same family:

    • naturality of phi_lambda;
    • diagonal vanishing on T;
    • polar vanishing on T x M;
    • radical containment for the symmetrized polar form.

    The conversion to the current actual packet is theorem-level. The only algebra used in the conversion is the proven extraspecial square/polar API from MinimalBlock.lean: extraspecial_annihilates and extraspecial_radical_subset.
