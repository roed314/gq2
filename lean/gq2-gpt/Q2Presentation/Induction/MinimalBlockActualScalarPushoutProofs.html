    Actual scalar central-pushout proof split #

    This sidecar strictly narrows the current Section 7 residual minimalBlock_preheadActualScalarCentralPushoutData.

    Manuscript anchor: Section 7, pages 41--42. For every nonzero invariant character lambda : R -> F_2, the scalar pushout K_lambda = K / ker lambda is a central double cover of M = K/R; its square map is the extraspecial square attached to the same phi_lambda.

    The current Lean API already has a standard same-family square/polar packet MinimalBlockPreheadCentralPushoutStandardData. The only remaining pushout-specific content needed to recover the actual-scalar packet is the comparison that this standard square is the extraspecial square. Surjectivity of every nonzero lambda : R -> F_2 is closed below as an elementary ZMod 2 fact.
