    Route12 central-pushout residual refinement #

    This sidecar leaves the routed files untouched. It refines the two route12 Section 7 residuals

    • minimalBlock_preheadCentralPushoutSquarePolarData;
    • minimalBlock_preheadCentralPushoutSimpleHeadDeterminantActualData;

    into one same-family packet: an actual scalar central-pushout square/polar family, plus the simple-head determinant pieces for the semantic family induced by that exact scalar pushout.

    The route-ready theorem exported below has the same lower-pushout conclusion used by IntegratedCurrentRouteProofs12, so integration only needs an import and a theorem-name swap in the route file.
