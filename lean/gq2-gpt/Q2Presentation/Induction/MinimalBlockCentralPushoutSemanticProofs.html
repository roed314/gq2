    Actual scalar central-pushout proof boundary #

    This sidecar narrows the current Section 7 semantic residual minimalBlock_preheadCentralPushoutSemanticData.

    The manuscript construction on pages 41--42 starts with scalar central pushouts K_lambda = K / ker lambda for nonzero invariant characters lambda : R^vee. The current semantic packet only records the resulting self-duality family phi_lambda. The residual introduced here is narrower: it records the scalar quotient condition for every nonzero lambda, the square map of the same scalar pushout family, and the fact that this square is the extraspecial form attached to the same phi_lambda.

    No routed file imports this sidecar yet. A coordinator can route through minimalBlock_preheadCentralPushoutSemanticData_proven to replace the broad semantic residual by minimalBlock_preheadActualScalarCentralPushoutData.

    Actual scalar central-pushout data over a pre-head Section 7 tower.

    Manuscript anchor: for each 0 != lambda in the determinant character space, the scalar quotient lambda : R -> F_2 forms K_lambda = K / ker lambda. Its square map on M = K/R is the extraspecial quadratic form determined by the same self-duality pairing phi_lambda.

    The current API has no quotient group object for K_lambda; this packet is the closest available Lean boundary to the manuscript construction without inventing dummy pushouts.
