    Actual semantic central-pushout closers #

    This sidecar is the narrow bridge below minimalBlock_preheadCentralPushoutExtraspecialActualData.

    The current route-facing packet combines two different manuscript steps:

    • the actual scalar central-pushout family K_lambda = K / ker lambda, recorded in Lean as one coherent square/polar family with the same phi_lambda;
    • the simple-head determinant theorem for that actual family, namely naturality, q_lambda|T = 0, b_lambda(T,M) = 0, and polar-radical containment.

    This file keeps those steps separate. It does not introduce a new axiom: the assembler below uses the existing concrete square/polar central-pushout packet and the existing actual simple-head determinant packet for the semantic family projected from that same square/polar data.
