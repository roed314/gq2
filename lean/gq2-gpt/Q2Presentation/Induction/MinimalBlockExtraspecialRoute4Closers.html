    Route4 extraspecial/determinant closers #

    This sidecar keeps PresentationCorrect untouched. It replaces the seven current route3 Section 7 central-pushout residual targets by projections from one actual scalar central-extension pushout family:

    • minimalBlock_preheadCentralPushoutQuadraticFamilyData;
    • minimalBlock_preheadCentralPushoutSquareDiagonalData;
    • minimalBlock_preheadCentralPushoutPolarSymmData;
    • minimalBlock_preheadCentralPushoutNaturalitySemanticData;
    • minimalBlock_preheadCentralPushoutDiagonalZeroSemanticData;
    • minimalBlock_preheadCentralPushoutPolarZeroSemanticData;
    • minimalBlock_preheadCentralPushoutRadicalContainmentSemanticData.

    The route4 packet below is backed by the split actual K_lambda = K / ker lambda construction packets from MinimalBlockActualCentralExtensionPushoutProofs. Thus the square/polar and determinant projections all use the same actual scalar pushout family.
