    Proof/refinement sidecar for the actual Section 7 group-invariant packet #

    This file does not edit the routed capstone. It records two theorem-level interfaces around the remaining final-closure packet minimalBlock_groupInvariantActualData.

    First, the current aggregate packet is reassembled from the existing lower group/module, simple-head quotient, and lower scalar quotient APIs. This proves that the aggregate is not logically stronger than the already split same-tower interfaces.

    Second, the route below uses a strictly smaller coherent packet. The current framed route only consumes:

    • the chosen lower subgroup/module tower;
    • dual-invariant vanishing on the raw module from that tower;
    • a raw nonzero invariant determinant character on the same Frattini layer.

    It does not consume the stronger quotient-head packet or the stronger lower scalar quotient packet directly. The smaller packet below keeps those consumed facts attached to one selected lower tower and routes the final framed-count argument without using minimalBlock_groupInvariantActualData.
