    Route39 minimal-block kernel/head refinement #

    This sidecar is intentionally standalone: no routed or root files import it yet.

    Route38 exposes the Section 7 kernel/head frontier through split concrete packets plus four Route38 raw-head packets. Route39 exports a drop-in group tower provider whose axiom closure avoids those Route38 names. The replacement frontier is:

    • the route26 upper-candidate leastness packet for the selected K;
    • one manuscript lem:collapse packet for the literal Phi(K), with normality proved from characteristicity and nontriviality lowered to a nonidentity witness;
    • the finer raw quotient transport packets for R, M, T, C, the induced actions, T-stability, the literal K/Phi(K) ~= M bridge, and a nonzero head coset;
    • the existing same-raw-tower simple-head/scalar actual closer.

    structure Q2Presentation.Induction.MinimalBlockKernelConcreteRoute39.FrattiniCentralizerCollapseData {p : FramedPair} {chief : NonScalarChiefFactor p} (K : MinimalBlockKernelLeastOverChiefData chief) :
    Prop

    Route39 residual for the hard finite-group content of manuscript lem:collapse, attached to the already selected least kernel K.

    Normality of Phi(K) in Y is theorem-level from K.Normal, so this packet only keeps the nonidentity witness and the pointwise collapse conclusions: centrality in K, K^4 = 1, and exponent two on Phi(K).

    • frattini_nonidentity : ∃ r ∈ Subgroup.map K.Ksub.subtype (frattini ↥K.Ksub), r ≠ 1
    • frattini_central_in_K (r : p.fst.Y) : r ∈ Subgroup.map K.Ksub.subtype (frattini ↥K.Ksub) → ∀ k ∈ K.Ksub, r * k = k * r
    • K_fourth_eq_one (k : p.fst.Y) : k ∈ K.Ksub → k ^ 4 = 1
    • frattini_square_eq_one (r : p.fst.Y) : r ∈ Subgroup.map K.Ksub.subtype (frattini ↥K.Ksub) → r ^ 2 = 1
