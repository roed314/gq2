    Route10 kernel/Frattini refinements for Section 7 #

    This sidecar is route-neutral: it does not edit the integrated route files. It targets the route10 P3 kernel/Frattini/raw residual cluster by replacing the independent concrete packet names with the coherent manuscript tower from Section 7:

    • the least normal K <= P over the selected non-scalar chief factor;
    • the Frattini-centralizer collapse for the literal R = Phi(K);
    • the raw quotient tower R, M = K/R, T, C, transported actions, the K/R ~= M bridge, and the nonzero head.

    The theorem-level candidate packet is proved directly from chief.upper. The leastness, collapse, and raw quotient tower remain explicit finite group/module inputs; this file keeps them coherent instead of using route10's arbitrary independent choices for each downstream field.
