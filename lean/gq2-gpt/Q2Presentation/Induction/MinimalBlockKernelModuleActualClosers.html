    Actual kernel-module closers for the current Section 7 route #

    This sidecar keeps the routed files untouched. It supplies route-ready wrappers for the current residual

    minimalBlock_kernelModuleActualProofs : Nonempty (MinimalBlockKernelModuleActualProofs chief)

    without using that aggregate axiom. The replacement route goes through the smaller manuscript-shaped packets already exposed by MinimalBlockKernelModuleConcreteClosers2:

    • least K over the chosen chief factor;
    • literal R = Phi(K) normal/nontrivial/central/fourth-power/exponent facts;
    • raw R, M, T, C objects and induced actions;
    • the literal K/R ~= M bridge;
    • nontriviality of the head M/T.

    The final theorem in this file mirrors the current integrated finite-surjection route, with the kernel-module aggregate residual replaced by those smaller packets.
