    Concrete kernel/module closers for the Section 7 subgroup tower #

    This sidecar is intentionally route-neutral. It replaces the five current concrete kernel/module residual names by one same-tower provenance packet.

    The packet is closer to the manuscript construction than the five independent residual calls: it carries one realized subgroup tower R = Phi(K) <= K <= L_Y, the Frattini-collapse facts from lem:collapse, the raw objects R, M = K/R, T, C, the induced actions, the literal K/R ~= M bridge, and the nonzero simple head.

    Concrete provenance for the Section 7 minimal-block kernel/module tower.

    This is the manuscript data behind the current five concrete residuals, bundled so downstream routes cannot mix a least K, a Frattini layer, raw module data, a quotient bridge, and a head proof coming from different towers.

    • Ksub : Subgroup p.fst.Y

      The least normal subgroup K <= P with K S = P.

    • Ksub_normal : self.Ksub.Normal
    • Ksub_le_upper : self.Ksub ≤ chief.upper
    • Ksub_le_LY : self.Ksub ≤ p.fst.LY
    • KS_eq_upper : self.Ksub ⊔ chief.lower = chief.upper
    • minimal_K (K' : Subgroup p.fst.Y) : K'.Normal → K' ≤ chief.upper → K' ⊔ chief.lower = chief.upper → self.Ksub ≤ K'
    • Rsub : Subgroup p.fst.Y

      The realized Frattini subgroup R = Phi(K) inside Y.

    • Rsub_normal : self.Rsub.Normal
    • Rsub_le_Ksub : self.Rsub ≤ self.Ksub
    • Rsub_card_gt_one : 1 < Nat.card ↥self.Rsub
    • Rsub_eq_frattini : self.Rsub = Subgroup.map self.Ksub.subtype (frattini ↥self.Ksub)
    • Rsub_central_in_K (r : p.fst.Y) : r ∈ self.Rsub → ∀ k ∈ self.Ksub, r * k = k * r

      lem:collapse: R centralizes K.

    • K_fourth_eq_one (k : p.fst.Y) : k ∈ self.Ksub → k ^ 4 = 1

      lem:collapse: K^4 = 1.

    • Rsub_square_eq_one (r : p.fst.Y) : r ∈ self.Rsub → r ^ 2 = 1

      lem:collapse: the Frattini layer is elementary abelian.

    • rawModuleData : MinimalBlockRawModuleData p chief

      The raw finite objects R, M, T, and C.

    • rawActionData : MinimalBlockRawActionData self.rawModuleData

      The induced C-actions on R and M, with T stable.

    • KmodR_equiv_rawM : ↥(Subgroup.map (QuotientGroup.mk' (Subgroup.map self.Ksub.subtype (frattini ↥self.Ksub))) self.Ksub) ≃* Multiplicative self.rawModuleData.M

      The literal quotient K/R realizes the abstract raw module M.

    • headNontrivialData : MinimalBlockHeadNontrivialData (MinimalBlockRawTowerData.ofModuleAction self.rawModuleData self.rawActionData)

      The simple head M/T is nonzero.
