    Finer concrete kernel/module closers for Section 7 #

    This sidecar is route-neutral and does not edit the routed proof files. It targets the five concrete Section 7 kernel/module packets currently exposed by MinimalBlockKernelModuleConcreteProofs:

    • minimalBlock_kernelModuleConcreteLeastData;
    • minimalBlock_kernelModuleConcreteFrattiniData;
    • minimalBlock_kernelModuleConcreteRawData;
    • minimalBlock_kernelModuleConcreteBridgeData;
    • minimalBlock_kernelModuleConcreteHeadData.

    The manuscript tower is kept fixed: choose the least normal K <= P with K S = P, set R = Phi(K), M = K/R, T = (K cap S)/R, and C = Y/K. Where the current Lean API has no literal subgroup quotient construction, this file splits the packet into smaller same-tower obligations rather than replacing it by an unrelated abstract module packet.
