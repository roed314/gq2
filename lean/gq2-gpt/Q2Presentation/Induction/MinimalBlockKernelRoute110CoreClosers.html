    Route110 kernel-core closers #

    This sidecar is isolated from the integrated route files. It provides a Route110-compatible kernel provider whose closure avoids the four broad kernel core atoms still visible in Route110:

    • minimalBlock_kernelFrattiniCollapseFullData;
    • minimalBlock_kernelLeastCandidateData;
    • minimalBlock_kernelLeastUniversalData;
    • minimalBlock_kernelRawQuotientTowerData.

    The replacement keeps the same Section 7 tower shape from the manuscript: choose the least K, set R = Phi(K), form the raw K/R, T, and C tower, and then use the same-tower simple-head data.
