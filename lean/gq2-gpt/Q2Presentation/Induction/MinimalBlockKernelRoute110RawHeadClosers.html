    Route110 raw-head closer sidecar #

    This sidecar keeps Route110's existing least/Frattini/raw-quotient kernel tower and changes only the same-tower simple-head assembly. The Route106 kernel provider obtained that assembly through Route63, whose closure exposes the four Route43 raw-head leaves. Here the same consumer packet is projected from the coherent same-raw-tower actual packet via Route83's rawSimpleHeadSemanticData_from_route83.

    No integrated route file imports this module yet.
