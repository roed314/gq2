    Route116 kernel closers #

    This sidecar leaves all integrated route files untouched. It sits below MinimalBlockKernelRoute114Closers and makes two route-neutral refinements of the Section 7 kernel provenance frontier:

    • the selected least kernel is built from the theorem-level upper chief-factor candidate, so the provider no longer depends on the residual MinimalBlockKernelRoute81Closers.leastKernelCandidateData atom;
    • the raw quotient tower is assembled directly through the Route43 literal transport packet, so the raw-tower theorem itself does not drag in the simple-head semantic atoms that are only needed later for the invariant packet.

    The final group-tower provider has the same consumer shape as the Route114 kernel provider, with a strictly smaller overall kernel axiom closure.
