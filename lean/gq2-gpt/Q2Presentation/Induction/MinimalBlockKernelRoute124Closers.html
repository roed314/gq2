    Route124 kernel closers #

    This sidecar leaves IntegratedCurrentRouteProofs124 untouched. It supplies a drop-in kernel provider with the same downstream shape as minimalBlock_groupTowerData_from_kernelCompactRoute120Closers.

    Route124 currently keeps the compact Route120 kernel surface:

    • minimalBlock_kernelLeastUniversalData;
    • minimalBlock_kernelFrattiniCollapseFullData;
    • minimalBlock_kernelRawQuotientTowerData;
    • minimalBlock_invariantActualProofs2Data.

    The theorem below replaces the first three lower-tower atoms by the existing coherent lower group/module packet

    • minimalBlock_lowerGroupModuleData,

    which ties the selected least kernel, Frattini subgroup, raw module/action tower, K/R bridge, and nonzero head to one Section 7 tower. The invariant packet is kept compact rather than split, since the available decompositions increase the visible closure.
