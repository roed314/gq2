    Route130 kernel closers #

    This sidecar leaves the promoted Route130 files untouched. It attacks the current kernel leaf

    • minimalBlock_provenanceSemanticData.

    The Route130 replacement below proves the same provenance-semantic packet from the stricter concrete-final provenance route:

    • concrete subgroup/module provenance for the selected K, R = Phi(K), M = K/R, T, C, and bridge data;
    • direct dual-invariant vanishing on the selected raw tower;
    • quotient-head simplicity on that same tower;
    • a raw nonzero invariant determinant character on the Frattini layer.

    It also exposes a drop-in Nonempty (MinimalBlockGroupTowerData p chief) provider with the same consumer shape as minimalBlock_groupTowerData_from_kernelRoute126Closers.
