    Route212 Frattini sidecar for the Section 7 kernel cluster #

    This file is route-neutral: it does not edit the current Route212 integrated proof. It gives a drop-in Section 7 kernel provider whose least-kernel and Frattini-collapse side avoids the Route199 field-level atoms by reusing the existing coherent least-kernel packet and the Route67 Lemma 7.2 packets. The raw quotient/simple-head side remains the current Route199 raw/head lowering.
