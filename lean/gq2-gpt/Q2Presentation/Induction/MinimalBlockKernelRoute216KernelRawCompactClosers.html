    Route216 compact Section 7 kernel/raw provider #

    This sidecar is route-neutral: it does not edit the integrated Route216 theorem.

    The manuscript's Section 7, pages 40--42, chooses one minimal normal subgroup K, sets R = Phi(K), forms the quotient tower Y -> B = Y/R -> C = Y/K, and proves the simple-head and Frattini-centralizer-collapse facts on that same selected tower. This file records that construction as one compact kernel/raw layer packet, then derives the same public provider shape used by the current Route216 baseline:

    Nonempty (MinimalBlockGroupTowerData p chief).

    The point of this sidecar is compact closure, not a wide pointwise lowering. It avoids both the old Route212 split leaves and the Route214/Route200 pointwise decomposition.
