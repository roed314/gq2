    Route54 minimal-block kernel/provenance sidecar #

    This file is route-neutral: it does not edit the current integrated route.

    Route53 exposes thirteen independent Section 7 kernel/raw leaves. Route54 replaces that surface by one coherent same-tower packet assembled from the existing Route44 manuscript packets:

    • least-kernel data via the Route26 upper-candidate leastness route;
    • one full Frattini--centralizer collapse packet for the selected K;
    • one raw quotient tower packet for R = Phi(K), M = K/R, T, C, actions, bridge, and head nontriviality;
    • one same-raw-tower simple-head/scalar semantic packet.

    The key compatibility point is the projection KernelRoute54Data.toRoute53ConsumedData: it rebuilds the exact consumed packet used by MinimalBlockKernelRoute53.groupTowerData_from_route53, but without calling the Route53 kernel/raw residuals.
