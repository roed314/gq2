    Route78 Frattini closers for the minimal-block kernel #

    Route77 exposes the minimal-block least/Frattini frontier through the ten Route72 split atoms. This sidecar keeps Route76's raw-head refinements, but replaces the Route72 least/Frattini surface by selected manuscript-shaped packets:

    • the least kernel is selected once, with its universal property attached to that selected candidate;
    • the Frattini nonidentity witness is selected once, with nontriviality attached to that same element;
    • the centralizer and fourth-power collapse steps use selected core plus Hall-image packets, avoiding the Route72 requirement to prove Hall-image data for every possible core;
    • the square/commutator containment is closed by choosing the literal embedded Phi(K) layer, leaving only the exponent-two implication from the collapse steps.

    The public provider at the bottom is a drop-in group-tower provider with the same target shape as minimalBlock_groupTowerData_from_kernelRoute76RawClosers.
