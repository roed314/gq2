    Route78 raw object/action closers for the minimal-block kernel #

    This sidecar is route-neutral. It keeps the current Route76 least/Frattini assembly, but replaces the Route77 raw object/action/head frontier by two same-tower manuscript packets.

    • rawObjectActionBridgeData records exactly the raw R,M,T,C objects, the transported actions, T-stability, and the K/Phi(K) ~= M bridge consumed by Route55. It deliberately does not expose the stronger Route43 literal-object equivalences, because the current kernel assembly does not consume them.
    • rawSimpleHeadSemanticData is the Route44 lem:simplehead interface on the same raw tower: invariant hyperplanes contain T, nonzero stable submodules of M/T are all of M/T, the head action is not pointwise trivial, and the lower Frattini layer has a nonzero invariant character.

    The exported provider has the same target as minimalBlock_groupTowerData_from_kernelRoute76RawClosers, so it can be used as a drop-in kernel provider in a later integrated route.

    Route78 same-tower raw object/action/bridge packet.

    This is the exact raw quotient surface consumed by Route55: raw objects, raw actions, and the concrete bridge tying the selected quotient K/Phi(K) to the raw module M. It is narrower than the Route43 literal-object surface for this route, because it does not ask for unused literal equivalences for R and C.
