    Route83 kernel closers for the minimal-block route #

    This sidecar keeps the Route82 least/Frattini and raw R,M,T,C tower assembly, but lowers the simple-head part of the current Route84 kernel frontier.

    Route82 obtains the Route44 simple-head semantic packet from four Route43 residual packets:

    • maximal invariant submodules contain T;
    • orbit spans on the quotient head;
    • a nontrivial head-action witness;
    • a selected lower invariant character.

    Route83 instead builds the same semantic packet from the coherent same-raw-tower actual invariant packet minimalBlock_invariantActualProofs2Data. This avoids the four Route43 raw-head residual names in the group-tower provider closure while preserving the same Route55 consumer interface.
