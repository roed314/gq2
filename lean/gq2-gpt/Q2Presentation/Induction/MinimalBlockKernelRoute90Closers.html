    Route90 kernel closers for the minimal-block route #

    This sidecar is isolated from the integrated routes. It provides a drop-in replacement for minimalBlock_groupTowerData_from_kernelRoute85Closers while removing the four current Route90 kernel residual names from the provider closure:

    • minimalBlock_invariantActualProofs2Data;
    • MinimalBlockKernelRoute84Closers.rawLiteralObjectActionTowerData;
    • MinimalBlockKernelRoute85Closers.frattiniCentralizerCollapseSectionData;
    • MinimalBlockKernelRoute85Closers.leastKernelSectionData.

    The replacement keeps the same Route55 consumer surface. The least-kernel and Frattini-collapse Section 7 packets are reassembled from the Route81 split packets; the literal R, M, T, C tower is reassembled from the Route43 literal object/action packets; and the simple-head packet is projected from MinimalBlockInvariantActualProofs3.
