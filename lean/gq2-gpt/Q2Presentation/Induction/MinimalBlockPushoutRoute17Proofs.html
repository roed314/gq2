    Route17 actual central-pushout / determinant sidecar #

    Route16 exposes two Section 7 pushout leaves:

    • minimalBlock_preheadCentralPushoutSemanticData;
    • minimalBlock_preheadCentralPushoutSimpleHeadDeterminantActualData.

    This sidecar replaces the route-relevant use of both leaves by one coherent actual central-extension pushout packet, itself supplied by the narrower split packets in MinimalBlockActualCentralExtensionPushoutProofs:

    • actual phi_lambda;
    • actual square map;
    • diagonal identity;
    • polar identity;
    • naturality;
    • annihilation on T;
    • radical containment.

    The bundled Route17 theorem has the same conclusion as Route16's lower pushout hook, so it can be used at the Route16 consumption site without changing any shared closer.
