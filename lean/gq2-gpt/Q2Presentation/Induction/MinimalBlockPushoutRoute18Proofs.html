    Route18 actual central-extension / pushout sidecar #

    Route17 replaces the old pushout/determinant leaves by seven actual central-extension packets:

    • actual phi_lambda;
    • actual square map;
    • diagonal and polar identities;
    • naturality;
    • annihilation on T;
    • polar radical containment.

    The current API still does not expose the quotient group object K_lambda = K / ker lambda from the manuscript. This sidecar therefore closes the seven Route17 packets for one coherent manuscript-shaped family built from:

    • the concrete central-pushout square/polar packet; and
    • the actual simple-head determinant packet for the semantic family projected from exactly that square/polar packet.

    Thus the route-ready theorem below depends on two sharper pushout residuals, not the seven Route17 actual-central-extension residuals.
