    Route22 minimal-block pushout/determinant close-out #

    Route21 split the former broad Section 7 actual central-extension pushout residual into seven same-family packets:

    • minimalBlock_preheadActualCentralExtensionPhiData;
    • minimalBlock_preheadActualCentralExtensionSquareData;
    • minimalBlock_preheadActualCentralExtensionDiagonalData;
    • minimalBlock_preheadActualCentralExtensionPolarData;
    • minimalBlock_preheadActualCentralExtensionNaturalityData;
    • minimalBlock_preheadActualCentralExtensionAnnihilationData;
    • minimalBlock_preheadActualCentralExtensionRadicalContainmentData.

    This sidecar gives a route22 hook that constructs the route21 split packet from the existing coherent extraspecial actual-pushout API in MinimalBlockExtraspecialClosers3. All route21 fields are projected from one actual scalar central-pushout/extraspecial family, so the resulting Phi and Q data cannot drift onto independent arbitrary witnesses.
