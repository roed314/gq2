    Route25 minimal-block pushout/determinant refinement #

    Route24's coordinator hook

    • minimalBlock_preheadPushoutRouteLowerData_from_route24ActualDeterminant

    has the desired lower-route output shape, but its proof bottoms out at the broad actual central-extension residual

    • minimalBlock_preheadActualCentralExtensionPushoutData.

    This sidecar keeps the same route-lower output shape while rebuilding the route23 same-family packet from narrower manuscript-shaped pieces:

    • an actual scalar central-pushout family, obtained from the standard square/diagonal/polar packets plus the scalar comparison packet;
    • the split prop:simpleheaddet pieces for that same projected semantic family: naturality, diagonal zero, polar zero, and radical containment.

    The result is route24-compatible for the nonterminal Fourier branch: it returns MinimalBlockPreheadPushoutRouteLowerData D, but its axiom closure no longer contains minimalBlock_preheadActualCentralExtensionPushoutData.
