    Route45 minimal-block pushout/determinant provider #

    Route44 reduces the Section 7 pushout branch to two actual central-pushout leaves:

    • minimalBlock_preheadCentralPushoutSquarePolarData;
    • minimalBlock_preheadCentralPushoutSimpleHeadDeterminantActualData.

    This sidecar removes both names from the route provider closure. The square/polar target is reconstructed from the split standard central-pushout packets (phi_lambda, q_lambda), diagonal, and polar-symmetry data. The determinant target is reconstructed from the same semantic family using the four semantic prop:simpleheaddet packets: naturality, diagonal-zero, polar-zero, and radical-containment.

    The resulting lower provider is integration-ready as a replacement for minimalBlock_preheadPushoutRouteLowerData_from_route44; no integrated/root route file imports this sidecar yet.
