    Route55 minimal-block pushout/determinant refinement #

    Route54 leaves one bundled non-local packet:

    • minimalBlock_preheadCentralPushoutActualDeterminantRoute54Data.

    This sidecar proves that packet from the existing split actual central-extension pushout packets. The split packets are exactly the manuscript-shaped Section 7 data around prop:simpleheaddet: the actual phi_lambda family, the actual square maps q_lambda, the diagonal and polar identities, naturality, annihilation on T, and polar-radical containment.

    No routed file imports this sidecar yet. The theorem minimalBlock_preheadCentralPushoutActualDeterminantRoute54Data_from_route55SplitPackets is the drop-in Route55 provider for the old Route54 leaf, with no use of the Route54 bundled axiom.
