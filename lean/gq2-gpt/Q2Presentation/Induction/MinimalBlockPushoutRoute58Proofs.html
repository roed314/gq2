    Route58 minimal-block pushout/determinant refinement #

    Route55 exposes the active Section 7 pushout cluster as seven split actual central-extension packets:

    • the actual phi_lambda family;
    • the actual square-map family;
    • diagonal and polar identities;
    • naturality;
    • annihilation on T;
    • polar-radical containment.

    This sidecar strictly reduces that cluster for the current Lean interface. Once phi_lambda is fixed, the normalized scalar pushout square can be taken to be the extraspecial square Extraspecial.extraspecialForm (phi_lambda). The square packet, diagonal identity, and polar identity are then theorem-level projections from PhiData; the remaining residuals are the actual construction of phi_lambda, naturality, annihilation, and radical containment for this normalized same-family square.

    No routed file imports this sidecar yet. The exported provider minimalBlock_preheadDeterminantPackets_from_route58 has the same downstream surface as Route55 while avoiding the three split square/diagonal/polar packet axioms.
