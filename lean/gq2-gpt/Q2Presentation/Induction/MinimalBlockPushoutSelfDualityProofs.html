    Central-pushout self-duality proof boundary #

    This sidecar closes the old phi-only Section 7 packet MinimalBlockPreheadCentralPushoutSelfDualityData from the current semantic central-pushout API.

    The present API still does not expose a literal normalized extension cocycle for the scalar central pushout. The narrowest available coherent source is MinimalBlockPreheadCentralPushoutSemanticData: one family phi_lambda, with the extraspecial square and polar maps induced from that same family by toQuadraticFamily, squareDiagonalData, and polarSymmData. The theorem below therefore replaces the old final-closure self-duality residual by that semantic central-pushout family, without introducing a dummy or zero pushout.
