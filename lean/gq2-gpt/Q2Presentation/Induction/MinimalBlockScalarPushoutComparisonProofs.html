    Scalar central-pushout comparison closers #

    This sidecar narrows the current Section 7 scalar central-pushout square/polar closure without editing routed files.

    Manuscript anchor: Section 7, pages 41--42. For a nonzero scalar character lambda : R -> F_2, the pushout K_lambda = K / ker lambda has square map q_lambda and polar form b_lambda. The existing API exposes this as either:

    • one coherent square/polar packet MinimalBlockPreheadCentralPushoutSquarePolarData, or
    • the current final-route split into a quadratic family, square diagonal, polar symmetry, and an extra comparison packet.

    The comparison packet is theorem-level from the square and polar identities: two QuadF2 objects with the same form and polar map are equal. Consequently the actual scalar-pushout packet can be recovered from the coherent square/polar packet alone, replacing the four independent final-closure packets by one manuscript-shaped residual.

    No routed file imports this sidecar yet.
