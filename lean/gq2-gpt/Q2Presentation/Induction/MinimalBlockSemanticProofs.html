    Proof-data route for Section 7 semantic minimal-block packets #

    This sidecar keeps routed files untouched. It records the useful next de-axiomatization boundary for the Section 7 semantic packets:

    • one realized group tower R = Phi(K) <= K <= L_Y, carrying the actual M = K/R, T, C, induced actions, simple-head facts, and invariant determinant character;
    • one central-pushout/extraspecial determinant family over the pre-head tower, with naturality, diagonal zero, polar zero, and radical containment tied to that same family.

    The theorem-level route below does not call the nine current semantic packet axioms. A coordinator can use it by supplying a single provider for MinimalBlockSemanticProofData, then reuse the existing Section 8 cover route.

    A single manuscript-shaped proof object for the Section 7 minimal block.

    The groupTower field is the actual subgroup tower: it contains K, R, R = Phi(K), K <= L_Y, the K/R module bridge, the induced C-actions, and the simple-head/character facts. The determinant fields are all tied to one central-pushout self-duality family over that tower's pre-head data.
