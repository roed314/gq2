    Actual simple-head/scalar closers for Section 7 #

    This sidecar is the compatibility layer between the actual same-tower Section 7 packet and the current four simple-head/scalar residual names used by MinimalBlockSimpleHeadSemanticClosers.

    It uses the existing actual packet minimalBlock_invariantActualProofs2Data, whose fields are attached to one raw tower:

    • invariant kernels contain T;
    • the quotient head is simple;
    • the quotient-head action is nontrivial;
    • the lower Frattini layer has an invariant character hitting 1.

    The semantic closer converts those direct fields into its four public residual interfaces without routing through the older orbit-span/nonzero-character packet names.
