    Square/polar central-pushout proof hooks #

    This sidecar closes the MinimalBlockPreheadCentralPushoutSquarePolarData conclusion without using the old combined residual minimalBlock_preheadCentralPushoutSquarePolarData.

    There are two route-ready sources:

    • a standard same-family determinant packet MinimalBlockPreheadCentralPushoutStandardData, whose fields are exactly a shared (phi_lambda, q_lambda) family plus the diagonal and polar identities;
    • the narrower actual scalar central-pushout packet, which records the scalar quotient and identifies its square with the extraspecial square of the same phi_lambda.

    No routed/root file imports this sidecar yet.
