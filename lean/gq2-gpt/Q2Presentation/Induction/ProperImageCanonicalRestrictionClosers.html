    Proper-image canonical restriction closers #

    This sidecar closes one more Section 8 proper-image residual in the final route without editing the target-tower sidecars.

    Manuscript anchors:

    • eq:targettower: the realized tower supplies B = Y/R, C = Y/K, and ker(B -> C) ~= K/R = M;
    • eq:Mstage / eq:Mdescent: proper-image subtraction uses the same B -> C map, then restricts it to J ∩ L_B -> L_C.

    TargetTowerRealizationClosers already constructs the elementary quotient tower through a packet that contains ker(B -> C) ~= M. The previous proper-image route rebuilt cardinality data from an arbitrary tower and had to call the residual axiom centralCover_middleKernelEquivData for that tower. Here we keep the quotient tower paired with the kernel equivalence produced by the target-tower construction, so the integrated route no longer needs that residual name. The remaining restriction content is still the explicit canonical restriction packet from ProperImageExactClosers.
