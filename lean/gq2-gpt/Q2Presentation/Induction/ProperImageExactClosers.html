    Proper-image exact closers #

    This sidecar refines the Section 8 proper-image packets from TargetTowerConcreteClosers.lean without editing existing files.

    Manuscript anchors:

    • eq:Mstage: partition the elementary lifts by exact image J < B;
    • eq:Mdescent: use the exact sequence 1 -> J ∩ M -> J ∩ L_B -> L_C -> 1;
    • proper-image subtraction: the exact-image kernel is the literal intersection J ∩ ker(B -> C), identified with J ∩ M.

    The file avoids the three residual axiom names introduced in TargetTowerConcreteClosers.lean by proving the literal-kernel packet from the existing middle-kernel equivalence, and by replacing the arbitrary restricted map/kernel packets with a canonical restriction witness.
