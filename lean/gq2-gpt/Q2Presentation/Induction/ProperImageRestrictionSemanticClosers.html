    Proper-image restriction semantic closers #

    This sidecar splits the remaining canonical restriction residual from ProperImageExactClosers.lean into the three semantic facts used in the manuscript proof of eq:Mdescent.

    For every proper exact-image subgroup J < B mapping onto C, the actual quotient map B -> C should restrict to a surjective map J ∩ L_B -> L_C, and the kernel of B -> C should lie in L_B. The kernel-equivalence and literal-intersection parts are already theorem-level in ProperImageExactClosers; this file isolates only those three restriction facts and reassembles the older canonical restriction packet from them.
