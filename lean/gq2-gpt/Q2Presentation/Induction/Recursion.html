    Stage H — de-axiomatizing the closed exact-image recursion (manuscript §6–§8) #

    This file proves, as a theorem, the exact statement of the axiom Q2Presentation.Induction.boundaryFramed_recursion_step of CountsEqual.lean (the terminal-∨-recursive dichotomy of thm:closedrecursion ∨ lem:scalarterminal), WITHOUT using that axiom. See the report at the end of the module for the precise decomposition of what is genuinely proven here versus what remains isolated as minimal documented bridge axioms.

    The genuinely-proven mathematical core is the finite Fourier inversion over 𝔽₂ of lem:fourier (eq:fourier, l.3866) — pure finite linear algebra — which is the arithmetic heart of the recursion eq:recursionR1 and the phase-cover transform. Everything structural (the dichotomy dispatch, the strict-decrease bookkeeping, the source-independent assembly of combine) is proven on top of it.

    Part 1 — Finite Fourier inversion over 𝔽₂ (manuscript lem:fourier, eq:fourier) #

    This is the genuine finite linear-algebra core. For a finite 𝔽₂-vector space W (the obstruction space) with character group Ŵ = Module.Dual (ZMod 2) W, and a map o : X → W from a finite index set, the number of x with o x = 0 is recovered from the partial counts m_χ = #{x : χ (o x) = 0} by Fourier inversion. The whole section is proven sorry-free; it is exactly the 𝔽₂ Poisson-summation identity that the manuscript invokes in prop:finalfourier and lem:phasecover.

      Part 2 — The closed exact-image recursion, with Fourier inversion made load-bearing #

      We now de-axiomatize the recursion of boundaryFramed_recursion_step. The single isolated input is the pre-Fourier obstruction realization of prop:finalfourier (eq:recursionR1) together with the proper-image / phase-cover recipe of lem:properimagesubtraction/lem:phasecover (eq:recursionR2–eq:phasecovertransform): for a non-terminal target the two sources realise the central-Frattini R-valued obstruction on a finite index set, with the per-character partial counts expressed through strictly-smaller exact-image counts by a single source-independent recipe partialN. The Fourier inversion eq:recursionR1 itself is PROVEN here (fourier_recursion_step, via Fourier.fourier_inversion); only the obstruction realization is axiomatic. See the closing report for the precise decomposition.

                Report — the precise decomposition of boundaryFramed_recursion_step #
