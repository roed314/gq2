    The manuscript proof and its decomposition #

    lem:scalarterminal has two ingredients, which this file separates into two minimal, atomic, documented residual inputs — replacing the monolithic boundaryFramed_scalarTerminal_equal — plus the genuinely-proven project-specific wiring.

    • (finite group theory, eq:terminalpullback) Schur–Zassenhaus splits the odd part off the wild kernel: there is a normal odd-order subgroup Ñ ◁ Y centralizing L_Y, meeting it trivially, with Q = Y/Ñ a finite 2-group, so Y ≅ H ×_{H₂} Q (TerminalReduction). This is pure standard finite group theory (no Q₂). The project-specific wiring — that the tame quotient H = Y/L_Y is genuinely a tame finite group ⟨s, t ∣ rconj t s = t²⟩ (extracted from the boundary frame α : 𝕋 ↠ H) — is PROVEN here (frame_tame_witness, via the topological generation of 𝕋_tame). The remaining abstract step (tame + scalar-terminal ⟹ the splitting) is isolated as the standard-group-theory axiom terminalSplit_of_scalarTerminal: it is textbook (Schur–Zassenhaus, which is in mathlib, plus the metacyclic odd 2-residual and the coprime-action stability [L,A]=[L,A,A]), isolated only because the coprime-action stability lemma is absent from mathlib v4.28.0. See its docstring for the proof sketch.

    • (source comparison, eq:terminalcompatibility) With the target reduced to the 2-group Q, the boundary-framed surjections of either source biject with maps g : Π → Q compatible with the marking, and the resulting count is the same for the two sources "by the fully marked isomorphism of their maximal pro-2 quotients". This is genuine local class field theory — the marked maximal pro-2 identification of Γ_A and G_{ℚ₂} plus the terminal exact-image bijection of eq:terminalcompatibility. It is isolated as the single minimal documented bridge axiom terminalMarkedPro2_countCompatibility. It is not derivable from the current marked pro-2 package alone: that package supplies the fully marked maximal quotient, but the code does not yet formalize the count-level reduction from a TerminalReduction target to the compatible Π → Q maps.

    Both residual axioms are strictly smaller and more atomic than the original monolith, and neither asserts the terminal equality directly: the group-theory axiom asserts only the target's structural splitting, and the bridge axiom asserts only the source comparison, given that splitting.     Part 1 — the Schur–Zassenhaus terminal splitting (genuine finite group theory) #

    def Q2Presentation.Induction.TerminalReduction (Yt : BoundaryFramedTarget) :
    Prop

    The terminal Schur–Zassenhaus reduction datum (manuscript lem:scalarterminal, eq:terminalpullback). For a scalar-terminal boundary-framed target the wild kernel's odd complement splits off: there is a normal subgroup Ñ ◁ Y of odd order with 2-group quotient Q = Y/Ñ, centralizing L_Y. Equivalently Y ≅ H ×_{H₂} Q with Q a finite 2-group: the terminal exact-image problem is governed entirely by the 2-group Q, hence (downstream) by the maximal pro-2 data of the source.

      STANDARD FINITE GROUP THEORY (isolated; Schur–Zassenhaus + coprime stability). The group-theoretic core of lem:scalarterminal (eq:terminalpullback): a finite group Y with a normal 2-subgroup L_Y, tame metacyclic quotient H = Y/L_Y (two-generated by s, t with rconj t s = t²), all of whose Y-chief factors inside L_Y are trivial H-modules, splits as Y ≅ H ×_{H₂} Q — equivalently there is a normal odd-order subgroup Ñ ◁ Y, centralizing L_Y, with 2-group quotient Q = Y/Ñ.

      PROOF SKETCH (manuscript l.4484–4510). The tame relation makes ⟨t⟩ ◁ H odd (finite_tame_tau_odd) with cyclic quotient H/⟨t⟩, so H is 2-nilpotent and its 2-residual N = O^{2}(H) = ker(H ↠ H₂) has odd order. Let P = π_Y^{-1}(N); then L_Y ◁ P with [P : L_Y] = |N| odd, coprime to the 2-power |L_Y|, so Schur–Zassenhaus (Subgroup.exists_right_complement'_of_coprime) gives an odd complement Ñ ≤ P. Coprime action on the trivial chief series gives [L_Y, Ñ] = 1 (the coprime stability lemma [G,A]=[G,A,A] for coprime A), whence P = L_Y × Ñ, Ñ is the characteristic Hall 2'-subgroup of P (so Ñ ◁ Y), and Q = Y/Ñ has order |L_Y|·|H₂|, a 2-group.

      This is entirely standard finite group theory. The required coprime-action stability lemma is formalized in Induction/CoprimeStability.lean; it is not local class field theory. The local-CFT content is isolated separately in terminalMarkedPro2_countCompatibility.
