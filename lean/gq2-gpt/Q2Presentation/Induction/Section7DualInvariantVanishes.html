    Dual-invariant vanishing (M^∨)^C = 0 (lem:simplehead, part 2) #

    The deepest simple-head fact, proved from ⊆-minimality and chief-ness alone — no equivariant identification of M/T with the chief factor is needed:

    Let φ : M →ₗ 𝔽₂ be C-invariant, and let J ≤ Y be the preimage in K of the multiplicative kernel of φ. Then J is Y-normal (invariance), Φ(K) ≤ J ≤ K, and J ⊔ S is a Y-normal subgroup between S and P. By chief-ness it is S or P:

    • if J ⊔ S = P, ⊆-minimality of K forces J = K, so φ = 0;
    • if J ⊔ S = S, then [Y, K] ≤ J ≤ S (the conjugation差 of any k ∈ K lands in the kernel because φ is invariant), and with P = K·S every chief-factor commutator [y, n] lands in S — the factor would be scalar, contradicting chief.nontrivial_action.

    Hence φ = 0 (towerDualInv_zero). Everything here is proved — no axioms.
