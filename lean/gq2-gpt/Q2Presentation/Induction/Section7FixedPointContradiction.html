    Fixed-vector setup for the lower branch of lem:collapse #

    The remaining lower branch has a normal subgroup P0 ◁ Y of 2-power index such that [P,P0] ≤ S. This file starts the fixed-vector contradiction by building the literal conjugation action on the chief quotient P/S and proving that P0 fixes it pointwise.
