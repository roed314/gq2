    Section 7 fourth-power kernel lowering #

    This route-neutral sidecar lowers Section7FrattiniCentralConcrete.KernelMinimalFourthPowerCandidateData to the manuscript's actual fourth-power-kernel mechanism.

    After R = Phi(K) is known to centralize K, the remaining fourth-power part of lem:collapse is to show that the kernel of the map k ↦ k^4 contains the Hall-commutator subgroup [K, Ñ]. The earlier fixed-vector branch already proves that normalClosure([K, Ñ]) cannot lie under the lower chief term for the same tame 2-power-index witness. Consequently any normal fourth-power kernel containing [K, Ñ] still maps onto the selected chief factor, and the corrected KernelMinimal.minimal_inside forces it to equal K.

    The file deliberately introduces no axioms and does not edit routed files. It provides standard-axiom bridge theorems that turn the lower, manuscript-shaped kernel data into:

    • KernelMinimalFourthPowerCandidateData;
    • K^4 = 1 for the corrected KernelMinimal;
    • the older MinimalBlockKernelModuleConcreteFourthPowerData packet;
    • the existing Frattini exponent/full Frattini packets when supplied with the already isolated square/commutator-layer input.
