    The odd Hall lift centralizes the Frattini layer #

    This file packages the immediate consequence of the previous collapse steps for the actual Frattini layer R = Φ(K) of the ⊆-minimal kernel.

    • kernelFrattini_le_lower gives R ≤ S;
    • tame_oddHall_lift_centralizes_lower gives an odd Hall lift Ñ centralizing S;
    • hence Ñ centralizes R;
    • since [K,R] ≤ R, the subgroup [K,R] also centralizes Ñ.

    The last statement is one of the two concrete premises for the Hall--Witt / three-subgroup argument that will show [K,Ñ] ≤ C_K(R).

    The odd Hall lift from the tame quotient centralizes the Frattini layer R = Φ(K), because R ≤ S = chief.lower.
