    Route141 Section 7 kernel raw closers #

    This attack sidecar leaves the public route and all existing Route43/Route138 files untouched. It targets the Route141 kernel raw frontier:

    • the Route43 literal R, M, T, C raw-object/action atoms;
    • the monolithic same-tower invariant packet minimalBlock_invariantActualProofs2Data.

    The literal Route43 structures are reassembled from Route141-owned lower field packets. The invariant packet is rebuilt through MinimalBlockInvariantActualProofs3, so the old invariant axiom is not in the new integrated theorem closure.
