    Route151 Section 7 kernel closers #

    This sidecar leaves the integrated route files and root imports untouched. It strictly lowers the Route150 Section 7 kernel frontier by replacing the selected least/Frattini atoms, Route141 raw object/action atoms, and the compact invariant packet with one coherent same-tower selected packet.

    The packet follows the manuscript Section 7 tower:

    • a selected minimal K over the first non-scalar chief factor;
    • the Frattini-collapse data for R = Phi(K);
    • the selected raw R, M = K/R, T, C objects/actions and quotient bridge;
    • the actual simple-head/lower-character invariant facts on that same raw tower.

    The exported providers project this packet back to the Route150-consumed kernel interfaces and then replay the Route150 finite-count route with only the kernel source swapped.
