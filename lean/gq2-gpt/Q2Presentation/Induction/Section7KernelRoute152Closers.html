    Route152 Section 7 kernel packet lowering #

    This sidecar leaves the integrated Route152 file untouched. It strictly lowers the current kernel leaf

    Section7KernelRoute151Closers.kernelRoute151SelectedPackageData

    by splitting the selected Section 7 packet into manuscript-shaped facts:

    • the selected minimal K over the first non-scalar chief factor;
    • the concrete Frattini layer R = Phi(K);
    • the raw R, M = K/R, T, C object/action bridge;
    • the same-raw-tower head action, direct dual-invariant vanishing, head simplicity, and raw lower invariant-character facts.

    The bridge below reconstructs the exact Route151 packet from these smaller facts, so supervisors can swap the single Route151 leaf for this lowered route without editing the existing integrated theorem in this worker turn.
