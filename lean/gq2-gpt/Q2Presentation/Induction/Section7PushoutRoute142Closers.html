    Route142 Section 7 pushout/determinant lowering #

    This sidecar is a Route141-compatible pushout replacement. Route141 currently routes the minimal-block determinant line through Section7PushoutRoute138Closers, whose closure still exposes the seven split central-pushout atoms

    • minimalBlock_preheadCentralPushoutQuadraticFamilyData,
    • minimalBlock_preheadCentralPushoutSquareDiagonalData,
    • minimalBlock_preheadCentralPushoutPolarSymmData,
    • minimalBlock_preheadCentralPushoutNaturalitySemanticData,
    • minimalBlock_preheadCentralPushoutDiagonalZeroSemanticData,
    • minimalBlock_preheadCentralPushoutPolarZeroSemanticData, and
    • minimalBlock_preheadCentralPushoutRadicalContainmentSemanticData.

    The project already contains MinimalBlockPushoutRoute88Closers, whose packet is exactly the manuscript-shaped replacement for those seven split atoms: one actual scalar central pushout and one actual simple-head determinant theorem for that same semantic family. This file exposes that provider under a Section 7 Route142 name and leaves all public routed files untouched.
