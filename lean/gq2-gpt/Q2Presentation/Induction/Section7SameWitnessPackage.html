    A single odd-Hall witness carrying both §7 packages #

    The earlier §7 files proved two useful facts with existentially chosen odd subgroups:

    • the Hall--Witt/three-subgroup argument puts normalClosure([K,Ñ]) inside K ∩ C_Y(Φ(K));
    • the tame quotient has a normal 2-power-index preimage P0 = L_Y · Ñ, and in the lower branch this P0 acts trivially on the chief factor.

    For the collapse branch these facts must use the same Ñ. The point of this file is that the Hall--Witt argument only needs Ñ to have odd order. We first prove that uniform form, then package it with the tame preimage witness from Section7TwoPowerAction.
