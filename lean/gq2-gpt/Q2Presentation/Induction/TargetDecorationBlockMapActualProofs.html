    Actual target decoration block-map construction #

    This sidecar strictly narrows the current Section 8 residual centralCover_targetDecorationBlockMapData.

    Manuscript anchor: §8, lem:decorationblock. Once the decoration kills the Frattini layer R, it has a canonical descent to the actual quotient K/R. The concrete target packet already identifies that quotient with the abstract module M. This file builds the descended linear map and proves its agreement with thetaY on K.

    The only residual left here is the semantic compatibility between the abstract C-action B.actM and conjugation on the realized quotient K/R; this is recorded as invariance of the canonical descended map, not as an arbitrary choice of block map.
