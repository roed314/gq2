    Target decoration-kernel proof packet #

    This sidecar narrows the remaining target-side decoration residual centralCover_targetDecorationKernelData.

    Manuscript anchor: §8, lem:decorationblock. The decoration kills R = Phi(K), descends to a C-invariant linear map on the elementary block M = K/R, and then vanishes because (M^vee)^C = 0.

    The current subgroup packet knows K/R ≃ M, but it does not expose the conjugation/action compatibility needed to construct the descended decoration map from thetaY. The residual below is exactly that missing compatibility: it supplies the descended block map, proves it is C-invariant, and records its agreement with the original decoration on K.
