    Route10 target projection/action proof sidecar #

    This file is a route10-owned sidecar for the four projection/action packets introduced by TargetKmodRProjectionSemanticClosers:

    • centralCover_targetLowerQuotientMapData;
    • centralCover_targetLowerActionEquivData;
    • centralCover_targetKmodRRepresentativeActionData;
    • centralCover_targetKmodRConjugationClassIndependenceData.

    The direct route-ready provider below avoids those four names. It uses the canonical same-tower lower projection and representative action, while closing same-coset independence from the realized K/R ~= M bridge rather than from a separate commutator packet.
