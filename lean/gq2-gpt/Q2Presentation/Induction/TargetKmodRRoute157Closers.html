    Route157 target K/R quotient/action closers #

    Route156's target branch reaches TargetKmodRRoute152Closers.centralCover_targetDecorationKernelData_from_route152Closers, whose non-local target frontier is the two Route28 atoms:

    • TargetKmodRRoute28BridgeProofs.centralCover_targetRoute28LowerActionEquivData;
    • TargetKmodRRoute28BridgeProofs.centralCover_targetRoute28RepresentativeActionData.

    This sidecar supplies a drop-in decoration-kernel provider with the same downstream target, but it does not call those Route28 axioms. Instead it uses the concrete lower projection/action facts isolated in Route151:

    • a map Y -> B.C;
    • pointwise fibres for surjectivity;
    • the two kernel inclusions identifying its kernel with K;
    • the projected conjugation action of B.C on K/R.

    The first bridge below also shows how those lower projection facts imply the Route28 lower-action surface for any Route28 lower quotient map. The second bridge proves the Route28 representative-action surface for the Route157 selected projection. The main provider uses the still lower projected-action path directly, so its axiom frontier avoids both Route28 target leaves and the broader Route146 quotient/action leaves.
