    Route20 actual same-tower target K/R provider #

    Route19's target experiment routed through the older bridge surface. This sidecar stays on the actual same-tower API used by the route18 integration, but splits the target obstruction along the manuscript tower

    Y -> B = Y/R -> C = Y/K, with M = K/R.

    The concrete subgroup packet already contains R <= K, normality/Frattini data, and K/R ~= M. It does not contain the two missing compatibility facts:

    • the abstract lower action group B.C is the literal quotient Y/K;
    • the abstract action B.actM is the conjugation-induced action on K/R.

    Route20 records exactly those two facts and rebuilds the current target hook from them. The second fact is tied to the selected quotient identification, so it is weaker than the older all-projections ambient-action residual.
