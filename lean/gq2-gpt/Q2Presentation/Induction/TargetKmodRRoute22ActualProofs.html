    Route22 actual bridge for the target K/R tower #

    Route21 reduced the target K/R branch to two packets:

    • an actual lower projection Y -> B.C with kernel K;
    • representative action data for the selected lower projection.

    This sidecar keeps route21 untouched, but supplies route22 replacements from a stricter same-tower interface. The remaining leaves are:

    • the canonical same-tower equivalence Y/K ~= B.C;
    • a representative action law for the canonical projection induced by that equivalence.

    The quotient projection, its surjectivity, and the kernel equality are then plain quotient bookkeeping. The upgrade from representatives to all ambient representatives is still the route21 theorem-level conversion, which uses the already closed inner-conjugation theorem.
