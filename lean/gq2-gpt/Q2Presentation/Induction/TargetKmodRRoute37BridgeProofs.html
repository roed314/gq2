    Route37 target K/R bridge sidecar #

    Route36's target provider routes through the bridge closers, whose remaining frontier is:

    • TargetKmodRSameTowerBridgeClosers.centralCover_targetLowerProjectionBridgeData;
    • TargetKmodRSameTowerBridgeClosers.centralCover_targetKmodRRepresentativeActionBridgeData.

    This sidecar does not assert those bridge packets as axioms. Instead it uses the literal manuscript lower quotient C = Y/K and asks only for:

    • the identification of that literal quotient with the abstract action group B.C;
    • the representative action law for the theorem-level representatives of classes in Y/K.

    The projection Y -> B.C, its kernel, the representatives over B.C, and the same-coset extension to all ambient representatives are then reassembled theorem-level. This is a strict lowering, not a full closure: the current MinimalBlock interface still stores B.C abstractly, so the manuscript identification B.C = Y/K remains real semantic data.
