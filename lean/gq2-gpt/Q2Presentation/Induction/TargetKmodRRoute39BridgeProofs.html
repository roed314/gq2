    Route39 target K/R bridge sidecar #

    Route38's target provider asks directly for:

    • a concrete projection Y -> B.C with kernel K;
    • the all-ambient projected conjugation action on K/R.

    This sidecar keeps the same downstream provider shape but lowers those two Route38 names to the literal manuscript tower data:

    • identify the abstract lower action group with the quotient Y/K;
    • prove the action law for the theorem-level representative of each class in that quotient.

    The projection Y -> B.C is then the quotient map followed by the identification, and the extension from representatives to all ambient elements uses the already closed theorem that inner conjugation by K is trivial on K/R.
