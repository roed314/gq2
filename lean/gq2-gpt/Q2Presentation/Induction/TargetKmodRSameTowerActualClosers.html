    Actual same-tower closers for the target K/R action #

    This sidecar keeps the current target K/R surface tied to the literal manuscript tower

    Y -> B = Y/R -> C = Y/K, with M = K/R.

    It does not construct that tower. Instead it records the strictly smaller pieces needed by TargetKmodRSameTowerSemanticClosers:

    • an equivalence from the literal lower quotient Y/K to the abstract action group B.C;
    • an all-representatives action law, avoiding a separate representative section;
    • the group-theoretic condition [K,K] <= R, stated pointwise, from which inner conjugation by K is theorem-level trivial on K/R.
