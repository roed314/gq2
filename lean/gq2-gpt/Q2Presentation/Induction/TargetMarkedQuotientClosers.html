    Marked canonical quotient closers #

    This sidecar refines the current target quotient/marked exactness layer used in the final Section 8 route.

    Manuscript anchors:

    • Section 7, eq:targettower: Y -> B = Y/R -> C = Y/K and M = K/R;
    • Section 8, eq:Mstage/eq:Mdescent: exactness of L_Y -> L_B -> L_C and the middle-kernel identification.

    The current quotient-child API records equivalences with the quotient groups, but it still does not state how the quotient targets' marked kernels are the images of the original marked kernel. Accordingly, this file replaces the coarser target residual names by smaller packets:

    • middle and lower canonical quotient targets are separate;
    • marked maps are restrictions of the ambient quotient maps, with agreement fields;
    • marked-kernel identifications are separate from the marked map construction;
    • the kernel-image equivalence is split into a hom and a bijectivity packet.
