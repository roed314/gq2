    Concrete target marked-quotient proofs #

    This sidecar closes the current presentation_correct target residuals as far as the existing framed-target API allows, without editing the routed files.

    The manuscript tower is

    Y -> B = Y/R -> C = Y/K,

    with marked-kernel restrictions L_Y -> L_B -> L_C. The current API does not construct quotient framed targets from a raw normal subgroup: it needs data that the quotient targets' marked kernels and decorations are the inherited ones. Accordingly the broad residuals are reduced to the narrower packets already matching that manuscript data:

    • middle/lower framed quotient targets;
    • actual marked restrictions of the ambient quotient maps;
    • the corresponding marked-kernel identifications.

    The route-ready theorem at the end mirrors the current actual-closer route, but uses these split packets instead of the three broad residual names.
