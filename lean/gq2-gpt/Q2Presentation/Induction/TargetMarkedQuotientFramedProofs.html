    Framed target marked-quotient proof packet #

    This sidecar lowers the four current Section 8 target-side residual packets in the final route to one coherent framed quotient packet.

    The current API has theorem-level quotient-map algebra once quotient children are supplied, but it does not construct quotient boundary-framed targets from a normal subgroup killed by both piY and the decoration thetaY. In particular, it has no field asserting the manuscript fact that the decorations vanish on the block K (lem:decorationblock). Therefore the fully framed quotient constructor is still a real residual.

    The packet below keeps the quotient children, inherited marked kernels, and the restricted marked maps tied to the same canonical quotient tower Y -> Y/R -> Y/K. Downstream projections are theorem-level.
