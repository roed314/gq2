    Semantic target marked-quotient closers #

    This sidecar attacks the current target marked/quotient residuals from TargetMarkedQuotientClosers.lean without editing routed files.

    Manuscript anchors:

    • Section 7, eq:targettower: choose R = Phi(K) <= K <= L_Y, set B = Y/R, C = Y/K, and identify M = K/R;
    • Section 8, eq:Mstage/eq:Mdescent: use the marked-kernel exact sequences L_Y -> L_B -> L_C and identify ker(B -> C) with M.

    What is proved here:

    • the split canonical middle/lower quotient packets are projections of the existing canonical quotient-target packet;
    • the split ker(B -> C) -> K/R hom and bijectivity packets are projections of the existing kernel-image equivalence packet;
    • the marked restriction packets are split into existing map/kernel packets plus new semantic agreement packets saying that the marked maps are the restrictions of the ambient quotient maps.

    The new agreement packets are the genuine remaining semantic content for the marked restrictions: the current API can construct a map L_Y -> L_B or L_B -> L_C and identify its kernel, but it does not yet say that the map is literally the quotient-map restriction.
