    Canonical target-quotient closers #

    This sidecar is intentionally not imported by the root module. It gives an alternate final-count route for the first current §8 priority: target/proper- image quotient bookkeeping.

    Manuscript anchors:

    • §7, eq:targettower: choose R = Phi(K) <= K <= L_Y, set B = Y/R, C = Y/K, and identify M = K/R;
    • §8, eq:Mstage/eq:Mdescent: use the marked-kernel exact sequences for Y -> B -> C and the literal J ∩ M proper-image kernels.

    The key point here is that the existing §7 group-tower packet already contains the realized subgroups R <= K, the Frattini identity, and the K/R ~= M bridge for the specific MinimalBlock built from that same packet. Therefore the target subgroup tower, Frattini, and K/R residual names are not needed in this route.
