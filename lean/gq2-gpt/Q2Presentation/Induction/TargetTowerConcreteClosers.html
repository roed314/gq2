    More concrete target-tower closers #

    This sidecar pushes the Section 8 target/proper-image packets one layer closer to the finite subgroup data used in the manuscript.

    Manuscript anchors:

    • eq:targettower: R = Phi(K), K <= L_Y, K/R ≃ M, and the quotient children Y/R, Y/K;
    • eq:Mstage / eq:Mdescent: the marked-kernel exact sequences and the literal J ∩ M kernels for proper exact-image subgroups;
    • lem:strictdecrease: proper exact-image kernels are proper in the elementary kernel because J < B but J -> C is onto.

    The file deliberately does not route the root theorem through these new packets. It supplies theorem-level reassembly lemmas from smaller semantic packets, so a future capstone reroute can replace the current target/proper-image residual names by these more concrete ones.

    General subgroup facts for the literal J ∩ M kernels #

    source
    theorem Q2Presentation.Induction.TargetTowerConcreteClosers.subgroup_eq_top_of_ker_le_of_surjective_restrict {G : Type u_1} {H : Type u_2} [Group G] [Group H] (f : G →* H) (J : Subgroup G) (hJ : Function.Surjective ⇑(f.comp J.subtype)) (hker : f.ker ≤ J) :
    J = ⊤

    If a subgroup maps onto the quotient and contains the kernel, then it is the whole domain. This is the finite-group reason that a proper exact-image subgroup J < B has J ∩ ker(B -> C) < ker(B -> C).
