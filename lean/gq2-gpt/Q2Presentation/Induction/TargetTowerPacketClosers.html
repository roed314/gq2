    Target-tower packet closers #

    This sidecar supplies theorem-level finite-group closers for the refined Section 8 target/proper-image packets in ElementaryTowerResiduals.lean.

    The current MinimalBlock interface still does not construct the realized normal subgroups R <= K <= L_Y inside an arbitrary target. Accordingly this file does not assert the five live packet axioms. Instead it proves the quotient-map and exact-sequence/cardinality parts once the missing concrete subgroups, framed quotient targets, and restriction maps are supplied as data.

    Manuscript anchors:

    • eq:targettower: Y -> B = Y/R -> C = Y/K;
    • eq:Mstage / eq:Mdescent: the exact sequences on marked kernels for proper exact-image subgroups.
