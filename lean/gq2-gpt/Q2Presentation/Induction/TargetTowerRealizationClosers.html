    Target-tower realization closers #

    This sidecar further refines the five concrete target-tower residuals from TargetTowerConcreteClosers.lean.

    Manuscript anchors:

    • Section 7, eq:targettower: choose R = Phi(K) <= K <= L_Y, with B = Y/R, C = Y/K, and M = K/R;
    • Section 8, eq:Mstage/eq:Mdescent: the marked-kernel exact sequences L_Y -> L_B -> L_C and the identification of ker(B -> C) with M.

    The current MinimalBlock interface is module-level. It does not, by itself, construct the realized subgroups, quotient framed targets, or marked-kernel restriction maps. Accordingly this file does not manufacture those objects. It splits the five concrete residuals into smaller semantic packets and proves the reassembly theorems whose closures no longer mention the five current concrete residual names.

            Type 1

            Canonical quotient targets attached to concrete subgroup data.

            This is the owned-file stand-in for the quotient-target constructor suggested by the Pro consult. The current global BoundaryFramedTarget API has no quotient constructor, and quotienting the decoration requires explicit factorization data, so the canonical quotient targets remain data here. Unlike CentralCoverTargetQuotientChildConcreteData, this packet does not duplicate the Frattini or K/R ~= M provenance: it only supplies the framed quotient targets equivalent to Y/R and Y/K.

            • middleChild : FramedPair

              The framed middle target canonically equivalent to Y/R.

            • lowerChild : FramedPair

              The framed lower target canonically equivalent to Y/K.

            • middleEquiv : p.fst.Y ⧸ D.Rsub ≃* self.middleChild.fst.Y

              The middle target realizes the quotient group Y/R.

            • lowerEquiv : p.fst.Y ⧸ D.Ksub ≃* self.lowerChild.fst.Y

              The lower target realizes the quotient group Y/K.
