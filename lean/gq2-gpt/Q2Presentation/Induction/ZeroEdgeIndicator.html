    def Q2Presentation.Induction.CochainSolvable (Γ : ProfiniteGrp.{0}) (F : ↑Γ.toProfinite.toTop → ↑Γ.toProfinite.toTop → ZMod 2) :
    Prop

    Cochain solvability (design §4-N2): the 2-cochain equation of F on the profinite source Γ has a continuous (= locally constant, Γ profinite and 𝔽₂ discrete) solution — ∃ c : Γ → 𝔽₂ continuous with c (γγ') = c γ + c γ' + F γ γ'. This is the SOLVABILITY INDICATOR of the normalized-cochain house style: no cohomology carrier is ever formed, and F need not be a cocycle for the Prop to be well-formed (the calculus below restricts to cocycles exactly where the mathematics does).
