    structure Q2Presentation.Lifting.CocycleExt (Yt : BoundaryFramedTarget) (zeta : Yt.Y → Yt.Y → ZMod 2) (hz1 : ∀ (y : Yt.Y), zeta 1 y = 0) (hz2 : ∀ (y : Yt.Y), zeta y 1 = 0) (hzc : ∀ (a b c : Yt.Y), zeta a b + zeta (a * b) c = zeta b c + zeta a (b * c)) :
    Type

    The carrier of the central extension Yt.Y ×_ζ 𝔽₂ determined by a normalized 2-cocycle ζ (manuscript l.4371–4372). A structure rather than a product-type synonym, so the twisted group instance below is canonical (keyed on the full cocycle datum) and cannot collide with the product instances; the normalization and cocycle laws are phantom parameters.

    • base : Yt.Y

      The base coordinate.

    • scal : ZMod 2

      The scalar (deck) coordinate.
