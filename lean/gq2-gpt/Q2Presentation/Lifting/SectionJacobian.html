    def Q2Presentation.Lifting.expSum (g : Gen) :
    GExpr → ZMod 2

    Mod-2 exponent sum of a generator in a word, with ω₂-powers counted once (the residue length is odd).

    Equations
    • Q2Presentation.Lifting.expSum g Q2Presentation.GExpr.one = 0
    • Q2Presentation.Lifting.expSum g (Q2Presentation.GExpr.gen a) = if a = g then 1 else 0
    • Q2Presentation.Lifting.expSum g (u.mul v) = Q2Presentation.Lifting.expSum g u + Q2Presentation.Lifting.expSum g v
    • Q2Presentation.Lifting.expSum g u.inv = Q2Presentation.Lifting.expSum g u
    • Q2Presentation.Lifting.expSum g (u.powNat k) = ↑k * Q2Presentation.Lifting.expSum g u
    • Q2Presentation.Lifting.expSum g u.powOmega2 = Q2Presentation.Lifting.expSum g u
