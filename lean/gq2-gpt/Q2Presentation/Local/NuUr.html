    The unramified character ν_ur : D₀ ⟶ ℤ₂ and the global Demushkin relator #

    (Stage: Local)

    This file equips the profinite Demushkin group D₀ (Boundary/D0.lean) with the two pieces of internal data needed to assemble the marked dyadic Demushkin normalization MarkedDemushkinQ2 (Local/MarkedDemushkin.lean), both proved sorry-free:

    • d0_globalRel : d0A ^ 2 * d0S ^ 4 * commutatorW d0S d0Y = 1 — the single defining Demushkin relator, holding globally in D₀ (not merely coordinatewise as in D0_rel). Obtained from D0_rel/D0_ext by pushing the genuine monoid hom θ_X = d0EvalAt X ∘ d0Iso.hom through the word.

    • nu_ur : D₀ ⟶ ℤ₂, the unramified character with the full marks ν_ur(a, s, y) = (-2, 1, 0) (manuscript Prop markedDem, eq:markedstandardhyp; here s is the geometric-Frobenius lift ↦ 1, a ↦ -2, y ↦ 0). This is the exact analogue of nu_2 : PiGrp ⟶ ℤ₂ (Boundary/PiGroup.lean) and nu_t : Ttame ⟶ ℤ₂ (Boundary/Ttame.lean): it is built as the map into ℤ₂ = lim_n ZMod(2ⁿ) whose n-th layer projects D₀ through an auxiliary cyclic Demushkin 2-group quotient cyclicD0 n (carrier C2Pow n, marks a = ofAdd(-2), s = ofAdd 1, y = 1). The marks (-2, 1, 0) are exactly consistent with the abelianized relator 2·(-2) + 4·1 = 0, so such a character exists.

    Note that this nu_ur is some character realizing the marks; it is not pinned to the genuine unramified character of G_{ℚ₂} (the interface does not require that — see Local/LabuteClassification.lean).

    Instance gotcha (inherited from Boundary/D0.lean, Boundary/PiGroup.lean) #

    Fin n carries the global Fin.instMul, not the transported groupFin. We thread groupFin explicitly (@, letI) through every Fin-level construction so Fin.instMul is never synthesized.

    The unramified character ν_ur : D₀ ⟶ ℤ₂ #

    The unramified character sends s to the topological generator 1 ∈ ℤ₂, a to -2, and y to 0. It is built as the map into ℤ₂ = lim_n ZMod(2ⁿ) whose n-th layer is the projection of D₀ through the auxiliary cyclic Demushkin 2-group quotient cyclicD0 n (carrier C2Pow n, a = ofAdd(-2), s = ofAdd 1, y = 1).
