    One-sided profinite reconstruction #

    This file is a self-contained, reusable module of profinite group theory. It is mathematically independent of the Q₂ word: it formalises the one-sided reconstruction lemma used at the very end of the main proof.

    The headline result is profinite_reconstruction_of_surj_counts: if P is a topologically finitely generated profinite group, Q is any profinite group, and P and Q admit the same number of surjective continuous homomorphisms onto every finite group, then P ≅ Q.

    Faithfulness note on the cardinality #

    The manuscript states the hypothesis as |Sur(P,H)| = |Sur(Q,H)| and its proof derives finiteness of Sur(Q,H) from finiteness of Sur(P,H) together with this equality (manuscript, proof of Lemma 2.5, first paragraph). We therefore use genuine cardinality Cardinal.mk rather than Fintype.card:

    • Fintype.card (Sur P (of G)) does not even elaborate, because there is no Fintype (P ⟶ of G) instance available a priori (finiteness needs P t.f.g., and the Q-side is not known finite before the argument runs); and
    • with Nat.card the theorem is false — take P trivial and Q an infinite-rank free profinite group: then Nat.card (Sur Q (ℤ/2)) = 0 = Nat.card (Sur P (ℤ/2)) for the wrong reason (the set is infinite), yet P ≇ Q.

    Cardinal.mk records the genuine cardinality, equals Fintype.card when a Fintype instance is present (Nat.card_eq_fintype_card/Cardinal.mk_fintype), and makes the one-sided lemma a true statement with only P finitely generated.

    Main statements #

    • SurjContHom — a surjective continuous homomorphism of profinite groups.
    • TopologicallyFinitelyGenerated — a finite set whose generated subgroup is dense.
    • tfg_hom_to_finite_finite — a t.f.g. profinite group has only finitely many continuous homomorphisms to a fixed finite group.
    • profinite_epimorphism_from_compatible_finite_quotients — a compactness / finite-intersection assembly of compatible finite quotient surjections.
    • tfg_profinite_hopfian — a t.f.g. profinite group is Hopfian.
    • profinite_reconstruction_of_surj_counts — the assembled reconstruction lemma.

            Hopficity #

            source
            theorem Q2Presentation.Profinite.tfg_profinite_hopfian (P : ProfiniteGrp.{u}) (hP : TopologicallyFinitelyGenerated P) (f : SurjContEnd P) :
            Function.Bijective ⇑(ProfiniteGrp.Hom.hom f.hom)

            A topologically finitely generated profinite group is Hopfian: a surjective continuous endomorphism is automatically bijective.

            The argument is the standard Hom-counting one. For a fixed finite quotient P ⧸ N, precomposition φ ↦ f ∘ φ is an injective self-map of the finite set Hom(P, P ⧸ N) (injective because f is surjective), hence surjective; so the quotient map factors through f. Evaluating that factorisation at a hypothetical nontrivial element of ker f (separated from 1 by some open normal N) gives a contradiction.

            Compatible finite quotients assemble to an epimorphism #

            source
            theorem Q2Presentation.Profinite.profinite_epimorphism_from_compatible_finite_quotients (A Q : ProfiniteGrp.{u}) (hfin : ∀ (N : OpenNormalSubgroup ↑Q.toProfinite.toTop), Finite (SurjContHom A (Q.diagram.obj N))) (hne : ∀ (N : OpenNormalSubgroup ↑Q.toProfinite.toTop), Nonempty (SurjContHom A (Q.diagram.obj N))) :
            Nonempty (SurjContHom A Q)

            From compatible finite quotient surjections to a global surjection. If, for every open normal subgroup N of Q, the set of surjections A ↠ Q ⧸ N is finite and nonempty, then the inverse limit of these finite nonempty sets is nonempty (compactness / finite intersection property), giving a surjection A ↠ Q.

            This is the topological core of the reconstruction lemma; see CategoryTheory.nonempty_sections_of_finite_cofiltered_system.

            The reconstruction lemma #

            source
            theorem Q2Presentation.Profinite.profinite_reconstruction_of_surj_counts (P Q : ProfiniteGrp.{u}) (hP : TopologicallyFinitelyGenerated P) (h : ∀ (G : Type u) [inst : Group G] [inst_1 : Finite G], Cardinal.mk (SurjContHom P (ProfiniteGrp.ofFiniteGrp (FiniteGrp.of G))) = Cardinal.mk (SurjContHom Q (ProfiniteGrp.ofFiniteGrp (FiniteGrp.of G)))) :
            Nonempty (P ≅ Q)

            One-sided profinite reconstruction. Let P be a topologically finitely generated profinite group and Q any profinite group. If for every finite group G the numbers of surjective continuous homomorphisms P ↠ G and Q ↠ G are equal (as cardinals), then P ≅ Q.

            The proof builds epimorphisms P ↠ Q and Q ↠ P from the equal counts (each finite quotient receives at least one surjection on the side that is its own quotient, and equal cardinality transports nonemptiness and finiteness across), and then uses Hopficity of the t.f.g. group P to upgrade P ↠ Q to an isomorphism.
