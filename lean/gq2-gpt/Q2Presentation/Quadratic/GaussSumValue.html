    Stage G / P6 — the Gauss-sum value G(Q) = ±2^{d/2} and agreement glue #

    prop:candidatezero (l.2731) and prop:localzero (l.3349) of reference/q2_manuscript.tex are zero-count statements; the §8 zero-edge formula (prop:zeroedge, eq:recursionR4b l.4342–4349) consumes them through the Gauss sum G(Q⁰) = ∑_x (-1)^{Q⁰(x)}: "the base local Gauss sum is negative in the unramified case and positive in the ramified case" (prop:localzero, l.3358–3360). This file supplies the value- and agreement-level glue, all proven (no axioms):

    • gaussType_eq_of_zeroCount — the Gauss type (Bool-valued Arf type of GaussSign.lean) is pinned by any realized zero count (extraction of the pinning step inside candidate_numZeros_eq_gaussType);
    • nonsingular_numZeros_eq_gaussType — the axiom-free Dickson restatement: #Q⁻¹(0) = baseZeroCount (gaussType Q) d for every nonsingular form with 0 < d (via nonsingular_numZeros_dickson);
    • gaussSum_eq_sign_two_pow — G(Q) = (-1)^{Arf(Q)}·2^{d/2} in the project spelling gaussSum Q = baseGaussSign (gaussType Q) * 2^(d/2);
    • two_mul_numZeros_eq — the unsigned bridge 2·#Q⁻¹(0) = 2^d + baseGaussSign (gaussType Q)·2^{d/2} (for ℕ-valued consumers of the §8 recursion formulas);
    • gaussSum_eq_of_numZeros_eq / gaussSum_eq_of_gaussType_eq — the common-G(Q⁰) glue: two nonsingular forms of the same (positive) dimension with the same zero count, or the same Gauss type, have the same Gauss sum. The still-missing §6 content is exactly the two type-pinning inputs (gaussType Q_A⁰ = gaussType Q⁰_loc = ramification type of V) — see audit/BLOCKR_P6_DESIGN.md for their status and provenance (prop:candidatezero = lem:wall + lem:unitaryline + lem:ramifiedhermitian; prop:localzero = local CFT + lem:shapirodet).

    Everything below is finite F₂ linear algebra plus integer arithmetic; proof idioms mirror the green Quadratic/DicksonCount.lean.

    theorem Q2Presentation.Quadratic.gaussSum_eq_sign_two_pow {V : Type u_1} [AddCommGroup V] [Module (ZMod 2) V] [Fintype V] (q : QuadF2 V) (hns : q.Nonsingular) (hpos : 0 < Module.finrank (ZMod 2) V) :

    The Gauss-sum value (prop:localzero l.3358–3360 reading: G(Q) = (-1)^{Arf(Q)}·2^{d/2}): for a nonsingular form on a finite space of positive dimension,

      gaussSum Q = baseGaussSign (gaussType Q) * 2^(d/2).

    Combines the Dickson count with the counting identity 2·#Q⁻¹(0) = 2^d + G.

The two §6 type-pinning inputs (gaussType Q_A⁰ = ram(V) from prop:candidatezero, gaussType Q⁰_loc = ram(V) from prop:localzero) plug in here to produce the single common factor G(Q⁰) of the zero-edge formula.
