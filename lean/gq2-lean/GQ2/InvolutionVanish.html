This file discharges hc2a from the concrete QuadraticAdjoin.exists_kummer_presentation (P-15f2c2a), leaving a version hvanish_involution_of_deepClass' that no longer carries the abstract package — only the tower (k ≤ L, hindex) and the unramifiedness hunram (the latter supplied by P-15f2c2c4's UnramifiedBridge.hunram_involution at the call site / f2d).
