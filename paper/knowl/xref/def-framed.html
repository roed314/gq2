Definition 4.1. Boundary-framed marked target.

A boundary-framed marked target is a tuple
\begin{equation*} \notn{calY}{\mathcal Y}=(\notn{Ytarget}{Y},\notn{LY}{L_Y},\notn{piY}{\pi_Y},\notn{thetaY}{\theta_Y}) \end{equation*}
where \(\notn{LY}{L_Y}\triangleleft \notn{Ytarget}{Y}\) is a finite \(2\)-group, \(\notn{piY}{\pi_Y}:\notn{Ytarget}{Y}\twoheadrightarrow \notn{Htarget}{H}\) has kernel \(\notn{LY}{L_Y}\text{,}\) and \(\notn{thetaY}{\theta_Y}:\notn{Ytarget}{Y}\to \notn{Etarget}{E}\) is a homomorphism. Put
\begin{equation*} \notn{qY}{q_Y}=(\notn{piY}{\pi_Y},\notn{thetaY}{\theta_Y}):\notn{Ytarget}{Y}\longrightarrow \notn{Htarget}{H}\times \notn{Etarget}{E}. \end{equation*}
For \(\notn{GammaSrc}{\Gamma}\in\{\notn{GA}{\GA},\notn{GQ}{\GQ}\}\) define
\begin{equation} \notn{eGamma}{e_\Gamma}^\beta(\notn{calY}{\mathcal Y}) =\#\{f:\notn{GammaSrc}{\Gamma}\twoheadrightarrow \notn{Ytarget}{Y}:q_Yf=\beta \notn{bGamma}{b_\Gamma}\}.\tag{4.4} \end{equation}
