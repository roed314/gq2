Lemma 8.2. Common scalar character group.

For both sources \(\Gamma\in\{\notnfar{GA}{\GA},\notnfar{GQ}{\GQ}\}\) one has
\begin{equation*} |\Hom_{\mathrm{cont}}(\Gamma,\F_2)|=8. \end{equation*}
Moreover scalar twisting by any such character preserves the boundary-framed condition for the central double covers used below.

Proof.
For the candidate, every map to \(\F_2\) kills \(\notn{tau}{\tau}\) because the tame relation forces the tame inertia generator to have odd order in finite quotients; hence it factors through the maximal pro-\(2\) quotient \(\notnfar{Pi}{\Pi}\text{.}\) For the local source, the same assertion follows from the characteristic wild subgroup and the common marked maximal pro-\(2\) quotient. The quotient \(\notnfar{Pi}{\Pi}\) has three mod-\(2\) generators, represented by \(\notn{sigma}{\sigma},\notn{x0}{x_0},\notnfar{x1}{x_1}\text{,}\) and one Demushkin relator with no linear mod-\(2\) part, so
\begin{equation*} H^1(\notnfar{Pi}{\Pi},\F_2)\cong\F_2^3. \end{equation*}
Thus the scalar character group has order \(2^3=8\) for both sources. The central covers used in the induction have central kernel killed by the fixed boundary and decoration, so multiplying a lift by a scalar character changes only the central coordinate and leaves the boundary-framed data unchanged.
