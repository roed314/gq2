Proposition 6.9. Candidate base determinant zero count.

If \(d=\dim \notnfar{Vhead}{V}\text{,}\) then
\begin{equation} \#(Q_A^0)^{-1}(0)= \begin{cases} 2^{d-1}-2^{d/2-1},&\notnfar{Vhead}{V}\text{ unramified},\\ 2^{d-1}+2^{d/2-1},&\notnfar{Vhead}{V}\text{ ramified}. \end{cases}\tag{6.33} \end{equation}

Proof.
We determine the unramified and ramified signs separately, using the Hermitian-line count in the first case and Lemma 6.8 in the second.
🔗
If \(\notnfar{Vhead}{V}\) is unramified, \(\notnfar{opU}{\mathsf U}=1\) and \(Q_A^0=q\text{.}\) Put \(F_V=\End_{H_V}(\notnfar{Vhead}{V})\cong\F_{2^{2m}}\) and let \(F_{V,0}\cong\F_{2^m}\) be the fixed field of the adjoint involution \(d\mapsto d^*\) defined by the polar form \(\notnfar{bq}{b_q}\text{.}\) The simple cyclic module \(\notnfar{Vhead}{V}\) is a one-dimensional \(F_V\)-space. The norm-one group
\begin{equation*} \mathcal U=\{u\in F_V^\times:uu^*=1\} \end{equation*}
preserves \(\notnfar{bq}{b_q}\text{.}\) For \(u\in\mathcal U\text{,}\) the function \(x\mapsto q(ux)+q(x)\) is therefore linear. It is \(H_V\)-invariant because \(u\) commutes with \(H_V\) and \(q\) is \(H_V\)-invariant. Since \(\notnfar{Vhead}{V}\) is nontrivial simple, \((\notnfar{Vhead}{V}^\vee)^{H_V}=0\text{,}\) so the function is zero. Thus \(\mathcal U\le O(\notnfar{Vhead}{V},q)\text{.}\)
🔗
By Lemma 6.7,
\begin{equation*} q(x)=\Tr_{F_{V,0}/\F_2}(a xx^*),\qquad a\in F_{V,0}^\times. \end{equation*}
The norm fibres have size \(2^m+1\text{,}\) so the number of zeros is \(2^{2m-1}-2^{m-1}\) and the Arf invariant is \(1\text{.}\)
🔗
Assume \(\notnfar{Vhead}{V}\) is ramified. The complete Hermitian and fixed-space calculation is Lemma 6.8; in particular \(\notn{Arf}{\Arf}(Q_A^0)=0\text{.}\) For a nonsingular quadratic form of dimension \(d\text{,}\) the identity
\begin{equation*} \#Q^{-1}(0)=2^{d-1}+(-1)^{\notn{Arf}{\Arf}(Q)}2^{d/2-1} \end{equation*}
now gives (6.33).
